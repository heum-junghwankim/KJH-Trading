# KJH-Trading

이치모쿠(메인)와 보조지표 5종을 한 화면에 올려, **추세 매매**와 **역추세 매매**를 같은 테이블로 판단하는 트레이딩뷰 Pine Script 모음입니다.

핵심은 "신호 하나를 믿는 것"이 아니라, **이치모쿠 신호 + 우측 상단 테이블 + 보조지표 포지션**이 한 방향으로 정렬됐는지 교차 검증하는 데 있습니다.

## 1. 지표 읽는 법

### 1-1. 이치모쿠 메인 신호 (8 / 22 / 44 / 22)

두 조건이 **동시에** 충족될 때 신호가 발생합니다.

| 신호 | 조건 |
| --- | --- |
| **매수(롱)** ▲ | 래깅 스팬(종가)이 기준선(22) 상향 돌파/이상 **+** 컨버전 라인(8)이 기준선 상향 돌파/이상 |
| **매도(숏)** ▼ | 위의 정확한 반대 |

- 같은 신호를 컨버전 **4 / 8 / 16** 세 스케일로 계산해 삼각형 **크기**(작음/중간/보통)로 동시 표시합니다.
- **스케일 캐스케이드**: 표시되는 매수 삼각형은 **4 → 8 → 16 컨버전 순서로만** 나옵니다(4가 먼저, 그다음 8, 그다음 16). 여러 크기가 한 봉/짧은 구간에 겹칠수록 시간축이 정렬된 것 → **신뢰↑**. 8/16까지 간 뒤 반대 신호가 나오면 사다리가 리셋됩니다.
- ⚠️ 추세추종이라 **횡보장 휩쏘 / 후행성**이 약점 → 신호만 믿지 말고 아래 테이블·보조지표로 검증합니다.

### 1-2. 우측 상단 테이블 — 각 행의 의미와 활용

| 행 | 값·의미 | 실전 해석 |
| --- | --- | --- |
| `상승/하락 전환` | 다음 봉 종가가 닿아야 매수/매도 신호가 되는 **트리거 가격**(4/8/16 세 스케일). 근접(±2%) 시 **둔탁한** 포지션 색 강조, 이미 진행 중이면 **선명** 포지션 색 `상승/하락 추세` | 신호 **직전 대기 가격**. 현재가가 트리거에 붙으면 진입 준비 |
| `평균 추세` | `횡보 압축 / 상승 분출 / 하락 분출` + 이벤트 후 가격 변화 %·**몇 봉째** | **압축**=휩쏘 경계(관망), **분출**=추세 시작 신호(봉수로 진행 정도 파악) |
| `비정상 매수/매도` | cRSI 과열+볼린저 이탈(고저) `비정상 매수/매도`, 이후 **반등 감시**(`롱/숏 감시(강)`). 둘 다 있으면 `비정상 매수 \| 숏 감시`처럼 병기, 정상이면 양봉=`정상 매수`·음봉=`정상 매도` | **역추세 진입 트리거**. 롱 감시=반등 매수 자리, 숏 감시=하락 반전 자리 |
| `이격도` | 종가÷MA×100(기준 100) · `되돌림 주의/확실/중립` | 과열/과매도 정도. **확실**이면 되돌림 임박 → 역추세 근거 |
| `되돌림 가격` | 이격도가 중립으로 복귀하는 목표가 | 역추세 매매의 **1차 목표가** 참고 |
| `구름대` | 지그재그가 구름 안에서 꺾일 때 **지지/저항 강도**(구름 두꺼울수록↑·접근 각도 완만할수록↑) | 지지/저항 **강함**이면 그 자리에서 반등/저항 기대 → 역추세 근거 |
| `VWAP 지지/저항` | VWAP 위=`지지`·아래=`저항` + `(이격도 \| 유지 봉수)`. 이격도=종가÷VWAP×100(기준 100), ±2σ 밖이면 둔탁색(과이격) | VWAP이 받침(지지)/누름(저항). 이격 과대면 되돌림 경계·봉수로 지속력 판단 |
| `VWAP 모양` | `기울기(상승/하락/평탄) \| 밴드폭(확장/수축/유지)` | **상승+확장**=추세 가속(초록), **하락+확장**=하락 가속(빨강), 수축=둔탁색·에너지 응축 |
| `VWAP 밴드 상태` | `하단/상단 밴드워킹`(추세) · `하단 지지/상단 저항`(되돌림) · `중앙(공방)` | 밴드워킹=추세 매매, 지지/저항·중앙=역추세/관망 |
| `장기 추세` | MA100·200 기반 `강한/약한 상승·하락 추세`·`골든/데드 크로스` | 큰 흐름. 추세 매매는 이 방향과 **같은 쪽**만 |
| `장기 지지/저항` | `100선 지지/저항 \| 200선 지지/저항`(몸통이 MA100/200을 완전히 이탈해야 지지/저항, 걸치면 중립) | 셀 배경: 둘 다 지지=파랑·둘 다 저항=빨강·엇갈림=둔탁(글자 흰색) |
| `롱/숏 스윕` | 샹들리에 트레일가 = 세력이 노리는 **스톱헌트(유동성) 자리** | 손절을 이 안쪽에 두면 잘 털림 → 스윕 **너머**로 |
| `롱/숏 청산` | 스윕 너머의 실제 **청산(손절) 가격** | 진입 시 손절 기준선. 라인·가격축 라벨로 표시 |
| `롱/숏 손익비` | 현재가 기준 **(목표 청산가까지)÷(손절 청산가까지)** `비율 (+목표%/-손절%)`. 현재가가 청산가 밖이면 `계산 불가` | **진입 타당성의 핵심 숫자**. 전략별 손익비 기준(아래)으로 필터 |

- 차트 오버레이: **평균 추세 다이아몬드**(압축=보라·상승 분출=파랑·하락 분출=빨강), **장기 추세 MA100/200 + backfill**, **스윕(점선)·청산(직선) 트레일**.

### 1-3. 보조지표 `포지션` 상태판

각 보조지표는 창 우측 상단에 **`롱 유리`(초록) / `숏 유리`(빨강) / `중립`(회색)** 을 한 가지로 표시합니다.

| 지표 | `롱 유리` | `숏 유리` | `중립`(관망) |
| --- | --- | --- | --- |
| 거래량 압력 | 평균선 상회 + **매수 우세** | 평균선 상회 + **매도 우세** | 평균선 하회(추진력 약함) |
| cRSI | 확장+과매도 / 상승 다이버전스 / 수축 후 상단 돌파 | 확장+과매수 / 하락 다이버전스 / 수축 후 하단 돌파 | 수축 중 밴드 안 / 중앙 |
| CCI | 과매도 뒤 골든크로스 → 동반 상승 | 과매수 뒤 데드크로스 → 동반 하락 | 무장 / 신호선 이탈 |
| OBV-ADX | ADX 충분 + `+DI 우위`(강화/약화) | ADX 충분 + `-DI 우위` | ADX 약함(휩쏘) |
| Auto VWAP | 상단 밴드워킹 / 하단 지지 | 하단 밴드워킹 / 상단 저항 | 중앙(공방) |

**정렬 = 신뢰도**: 5개 포지션이 모두 같은 방향이면 강한 신호, 거래량·OBV-ADX 포함 3~4개 일치면 보통, 엇갈리거나 `중립/휩쏘`가 섞이면 관망.

## 2. 매매 전략

두 전략은 **정반대의 시장 국면**을 노립니다. 진입 전 반드시 `VWAP 밴드 상태`와 `손익비`로 어느 전략의 자리인지부터 구분하세요.

### 전략 A — 롱숏 추세 매매 (손익비 1~2)

> **에너지가 응축(수축/중립)됐다가 분출하는 순간, 추세 방향으로 올라탄다.**

**국면 조건**
- `VWAP 밴드 상태` = **중앙(공방)** 또는 `VWAP 모양`의 밴드폭 = **수축** → 에너지 응축(스퀴즈). 이후 **확장**으로 전환되는 순간이 분출.
- `평균 추세` = **상승/하락 분출**(압축이 풀림).

**진입 체크리스트 (같은 방향으로 정렬될 때만)**
1. **이치모쿠 신호**: 진입 방향의 매수/매도 삼각형. 4→8 캐스케이드가 붙으면 신뢰↑.
2. **장기 추세** = 같은 방향(강한/약한 상승·하락 추세). 추세 매매는 **큰 흐름을 거스르지 않는다**.
3. **VWAP 모양** = 기울기 방향 일치 + 밴드폭 **확장**(초록/빨강 셀). `VWAP 밴드 상태`가 **밴드워킹**으로 바뀌면 추세 확정.
4. **보조지표 포지션** = 진입 방향으로 3개 이상(특히 거래량 압력·OBV-ADX). OBV-ADX가 `강화`면 가속 구간.
5. **손익비**: `롱/숏 손익비`가 대략 **1~2배**. 추세 매매는 높은 승률로 짧게 먹는 자리라 손익비가 크지 않아도 됩니다.

**관리**
- 손절 = 진입 방향의 `청산가`(스윕 너머). 스윕 안쪽에 두면 털립니다.
- 목표 = 반대편 `청산가` 또는 밴드워킹이 꺾일 때(`VWAP 밴드 상태`가 밴드워킹 이탈 / `VWAP 모양` 확장→수축).
- **청산 신호**: 반대 방향 신호, 밴드워킹 종료, 평균 추세 반대 분출 중 먼저 오는 것.

### 전략 B — 롱숏 역추세 매매 (손익비 3~10)

> **한쪽으로 과하게 쏠린 가격이 평균으로 되돌아오는 걸 크게 노린다. 승률은 낮아도 손익비로 이긴다.**

**국면 조건**
- `VWAP 밴드 상태` = **박스권**(상단 저항 / 하단 지지 반복, 중앙 공방) 또는 가격이 **잡으려는 포지션과 반대 방향으로 과도하게 이동**(밴드 바깥).
- `이격도` = **되돌림 주의/확실**(과열 또는 과매도).

**진입 체크리스트 (역방향 진입)**
1. **비정상 캔들 / 반등 감시**: `롱 감시`(하단 이탈 후 반등 자리) → 매수, `숏 감시`(상단 이탈 후 반전 자리) → 매도. 이 상태에선 **역 방향 신호가 더 빠르게** 나옵니다(래깅 위치 봉 종가 돌파 기준).
2. **이격도 확실** + `되돌림 가격`을 1차 목표로 참고.
3. **구름대 지지/저항 강함** 또는 **VWAP 상단 저항/하단 지지**에 가격이 닿음 → 반전 벽.
4. **보조지표 다이버전스/극단**: cRSI 과매수·과매도 + 다이버전스, CCI 극단 후 반대 크로스, OBV-ADX `약화`(추세 소진).
5. **손익비**: `롱/숏 손익비`가 **3~10배**. 극단에 붙어 진입해 손절이 타이트하고(가까운 청산가), 목표(반대편·평균 복귀)가 멀어 자연스럽게 높은 손익비가 나옵니다. **3배 미만이면 역추세는 진입하지 않습니다.**

**관리**
- 손절 = 극단 방향의 `청산가` 바로 너머(반전 실패 = 추세 지속이므로 빠르게 손절).
- 목표 = `되돌림 가격` → VWAP → 반대편 밴드로 **분할 청산**. 손익비가 큰 만큼 일부만 맞아도 이깁니다.
- **정상 캔들로 복귀**(이격도 중립)하면 되돌림이 끝난 것 → 남은 물량 정리.

### 전략 구분 요약

| 구분 | 추세 매매 (A) | 역추세 매매 (B) |
| --- | --- | --- |
| VWAP 밴드 | 중립/수축 → **확장·밴드워킹** | **박스권 / 반대 과이동**(밴드 바깥) |
| 진입 신호 | 방향 **매수/매도 신호** + 분출 | **비정상 캔들·반등 감시**(역 신호)·이격도 확실 |
| 장기 추세 | **같은 방향** | 무관/반대(되돌림) |
| 손익비 | **1~2** | **3~10** |
| 승률/보상 | 높은 승률·작은 보상 | 낮은 승률·큰 보상 |

## 3. 트레이딩 철학

- **신호 한 개는 가설, 정렬은 확신.** 이치모쿠 삼각형은 시작점일 뿐입니다. 테이블(구름대·추세·VWAP·이격도)과 보조지표 포지션이 **같은 문장을 말할 때만** 진입합니다.
- **먼저 국면을 정하고, 그다음 전략을 고른다.** `VWAP 밴드 상태`와 `손익비`로 지금이 **추세장(A)** 인지 **되돌림장(B)** 인지부터 판정하세요. 국면을 틀리면 좋은 신호도 손실이 됩니다.
- **손익비가 진입을 결정한다.** 추세는 1~2, 역추세는 3~10. 이 숫자를 만족하지 못하면 **아무리 신호가 좋아도 거른다.** 청산가(손절)와 목표를 먼저 정하고 손익비를 계산하는 습관이 곧 리스크 관리입니다.
- **세력의 스톱헌트를 피한다.** 손절은 항상 `스윕` 안쪽이 아니라 `청산가` 너머에 둡니다. 보이는 손절 군집은 털리라고 있는 자리입니다.
- **압축은 관망, 분출은 행동, 극단은 반전 준비.** `평균 추세` 압축 구간에선 정렬이 맞아도 신뢰를 한 단계 낮추고, 분출·비정상 캔들이 나올 때 계획대로 실행합니다.
