# 거래량 압력 추적

트레이딩뷰용 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine)

이 지표는 현재 봉의 `매수 우세 / 매도 우세`를 종가 위치 기반으로 추정해서, `비정상 거래량`과 `전체 거래량 평균선`을 함께 보는 간단한 압력 확인 도구입니다.

## 현재 소스 기준으로 보이는 것

| 요소 | 현재 표시 방식 |
| --- | --- |
| Buy Volume 추정치 | 봉의 종가가 고가에 가까울수록 크게 계산됩니다. |
| Sell Pressure 배경 막대 | 전체 거래량을 매도 계열 색으로 먼저 깔아 압력 바탕을 보여줍니다. |
| 비정상 거래량 색상 | 최근 평균 대비 강한 봉만 따로 강조합니다. |
| Total Volume SMA | 현재 거래량이 최근 평균보다 큰지 보는 기준선입니다. |

참고:
- 기본 비정상 거래량 기준은 `비교 캔들 수 = 20`, `비정상 배수 = 1.0`입니다.
- 즉 기본값에서는 최근 평균보다 `100%` 이상 큰 봉만 비정상으로 봅니다.

## 트레이딩 의미 핵심 3가지

### 1. 비정상 매수 압력 + 평균선 상회 의미

같은 봉에서 매수 우세와 비정상 거래량, 평균선 상회가 함께 나오면 `실제 매수 압력이 평소보다 강하게 붙은 상태`로 읽을 수 있습니다.

- 눌림 뒤 반등 확인용으로 가장 많이 씁니다.
- 메인 자리 신호와 겹치면 `롱 쪽 후속 확인` 의미가 커집니다.

요약하면, 이 조합은 `매수 힘이 실렸는지 확인`하는 용도입니다.

### 2. 비정상 매도 압력 + 평균선 상회 의미

같은 봉에서 매도 우세와 비정상 거래량, 평균선 상회가 함께 나오면 `실제 매도 압력이 평소보다 강하게 붙은 상태`로 읽을 수 있습니다.

- 반등 뒤 재하락 확인용으로 자주 씁니다.
- 저항 부근이나 기준 단가 재이탈과 겹치면 `숏 쪽 후속 확인` 의미가 커집니다.

요약하면, 이 조합은 `매도 힘이 실렸는지 확인`하는 용도입니다.

### 3. 방향은 있는데 평균선 아래일 때 의미

봉 색상이나 buy/sell 추정 방향은 있어도 평균선 아래에 머물면 `방향은 보이지만 실제 힘은 약한 상태`일 수 있습니다.

- 반응은 나와도 후속이 짧게 끝날 수 있습니다.
- 이 경우는 진입보다 `확인 대기` 쪽이 더 안전합니다.

요약하면, 평균선 아래 압력은 `신호는 있어도 추진력은 약할 수 있는 상태`입니다.

## 내부 로직

### 1. Buy / Sell 압력 추정

현재 코드는 아래 식을 씁니다.

- `buyVolume = volume * (close - low) / (high - low)`
- `sellVolume = volume * (high - close) / (high - low)`

즉 실제 체결 델타가 아니라, `봉 안에서 종가가 어디에 위치했는가`를 이용한 추정치입니다.

### 2. 비정상 거래량 조건

비정상 거래량은:

- `volume > sma(volume, 비교 캔들 수) * (1 + 비정상 배수)`

일 때만 켜집니다.

기본값은:
- `비교 캔들 수 = 20`
- `비정상 배수 = 1.0`

### 3. 평균선

`Total Volume SMA`는:

- `sma(volume, 평균선 봉 수)`

로 계산합니다. 이 선 위로 거래량이 올라오면, 현재 봉 압력이 `평소보다 실제로 큰지` 확인하는 데 쓸 수 있습니다.

## 우측 상단 상태판

차트 우측 상단에 현재 상태와 포지션 판정을 표로 띄웁니다.

맨 윗줄 제목 칸(`거래량 압력`)에 현재 `포지션` 값이 색상과 함께 표시됩니다.

| 행 | 내용 |
| --- | --- |
| 거래량 압력 (제목) | `포지션`: `롱 유리` / `숏 유리` / `중립` |
| 압력 방향 | 최근 N봉 매수량 합 vs 매도량 합의 비율로 `매수` / `매도` / `중립`. 예: `매수 (1.5 : 1.0)` |
| 평균선(SMA) | 최근 N봉 평균 거래량 vs 기준 평균선(20봉)의 비율로 `상회` / `보통` / `하회`. 예: `상회 (1.5 : 1.0)` |

`포지션`(제목 셀) 판정 — **평균선 상회(추진력 있음)일 때 현재 봉 압력 방향대로** 안내:
- `롱 유리`: 평균선 상회 + 매수 우세
- `숏 유리`: 평균선 상회 + 매도 우세
- `중립`: 평균선 하회 (방향은 있어도 추진력 약함 → 관망)

### 압력 방향 (2행) — 최근 N봉 매수/매도 압력 비율

- 최근 `N`봉(`압력/추진 비교 봉 수`)의 매수량 합(`sumBuy`)과 매도량 합(`sumSell`)을 구합니다.
- 비율 `R = max(sumBuy, sumSell) / min(sumBuy, sumSell)` (항상 1.0 이상).
- 판정:
  - `R ≤ k` → **중립** (양방향 압력이 엇비슷)
  - `R > k` 이고 매수 합이 크면 → **매수 우세**
  - `R > k` 이고 매도 합이 크면 → **매도 우세**
- 표기(소수 1자리): 큰쪽 값이 왼쪽/오른쪽 어디에 오는지로 방향을 드러냅니다.
  - 매수 우세: `매수 (R : 1.0)` — 왼쪽(매수)이 큰쪽
  - 매도 우세: `매도 (1.0 : R)` — 오른쪽(매도)이 큰쪽
  - 중립: `중립 (R : 1.0)` — 작은쪽을 1.0으로 둔 실제 비율
- 색: 매수 우세 = 녹, 매도 우세 = 빨강, 중립 = 회색. (강도는 아래 막대 색이 이미 표현하므로 방향만 순수 표기합니다.)

> **2행의 `1.0`은 "작은쪽 압력" 기준입니다.** 두 방향 압력 중 작은 쪽을 1.0으로 두고, 큰 쪽이 몇 배인지를 보여줍니다.

### 평균선(SMA) (3행) — 최근 N봉 평균이 기준 평균 대비 큰지

- 최근 `N`봉 평균 거래량(`recentAvg = sma(volume, N)`)을 기준 평균선(`baseAvg = sma(volume, 20)`)과 비교합니다.
- 비율 `ratio = recentAvg / baseAvg`.
- 판정:
  - `ratio > k` → **상회** (최근 거래량이 평소보다 확연히 큼)
  - `ratio < 1/k` → **하회** (최근 거래량이 평소보다 확연히 작음)
  - 그 외 → **보통**
- 표기(소수 1자리): **오른쪽은 항상 평균선 = `1.0` 고정**, 왼쪽에 `ratio` 원값. 예: `상회 (1.5 : 1.0)`, `보통 (1.1 : 1.0)`, `하회 (0.7 : 1.0)`.
- 색: 상회 = 녹, 보통 = 회, 하회 = 회 계열(약간 어둡게). **하회에 빨강을 쓰지 않는 이유는 "거래량이 적다"가 "매도"로 오독되는 것을 막기 위함**입니다.

> **3행의 `1.0`은 "기준 평균선" 자체입니다.** 2행의 `1.0`(작은쪽 방향 압력)과 의미가 다릅니다 — 2행은 방향 비교, 3행은 거래량 크기 비교입니다.

### 임계 k = 1.2 의 의미

`중립/보통 비율 임계 k`는 2행 중립 경계와 3행 상회·하회 경계를 공유합니다. `k = 1.2`는 **한쪽이 다른 쪽보다 20% 이상 클 때만** 방향(우세/상회·하회)으로 인정하고, 그 안쪽 차이는 중립/보통으로 봅니다. 잦은 노이즈성 방향 전환을 억제하는 최소 우위 폭입니다. (CEO 승인값)

## 자주 만지는 설정

| 설정 | 언제 조정하나 |
| --- | --- |
| `비교 캔들 수` | 비정상 거래량 기준을 더 짧게 또는 길게 보고 싶을 때 |
| `비정상 배수` | 비정상 판정을 더 엄격하게 또는 느슨하게 만들 때 |
| `평균선 표시` | 평균선 자체를 켜거나 끌 때 |
| `평균선 봉 수` | 현재 거래량 비교 기준을 바꿀 때 |
| `압력/추진 비교 봉 수 (N)` | 테이블 2·3행이 보는 최근 봉 수를 바꿀 때 (기본 5 = 평균선 20의 1/4) |
| `중립/보통 비율 임계 k` | 2행 중립·3행 상회/하회 경계를 더 넓게(둔감) 또는 좁게(민감) 만들 때 |
| 색상 설정 | 가독성에 맞춰 normal/abnormal 색을 바꿀 때 |

## 주의사항

- 실제 호가 체결 델타가 아니라 `봉 종가 위치 기반 추정치`입니다.
- 고가와 저가가 같은 봉은 buy/sell 추정값이 `0`으로 처리됩니다.
- 2행은 최근 N봉 매수/매도 합의 작은쪽이 `0`(예: 연속 high==low 구간)이면 비율을 낼 수 없어 `중립 (-)`으로, 3행은 기준 평균선이 `0`이면 `보통 (-)`으로 안전 표기합니다.
- 지표 하나만 보고 방향을 확정하기보다, 가격 구조와 다른 신호를 함께 보는 편이 안전합니다.
