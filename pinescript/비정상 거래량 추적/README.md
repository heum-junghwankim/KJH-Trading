# Buy/Sell Volume + Abnormal Volume + Alignment Background

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`main.pine`](./main.pine)

## 개요

이 지표는 거래량을 `Buy Volume`과 `Sell Volume`으로 나눠 보여주고, 평소보다 큰 거래량이 나왔는지 함께 표시합니다.

지금 버전에서는 여기에 `추세 정렬 배경`까지 추가되어, 단순히 거래량만 보는 것이 아니라 현재 시장이 상승 정렬인지 하락 정렬인지도 같이 볼 수 있습니다.

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`main.pine`](./main.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.
5. 필요하면 저장합니다.

## 예시 화면

![지표 예시 화면](./img.png)

위 이미지에서 볼 수 있는 핵심은 아래 3가지입니다.

- 흰색/회색 막대: 일반 매수/매도 거래량
- 연두색/빨간색 막대: 비정상 거래량이 붙은 매수/매도 구간
- 초록색/자주색 배경: 추세 정렬이 맞는 구간

쉽게 해석하면:

- `막대 색`은 거래량의 성격
- `배경 색`은 시장 방향의 정렬 상태

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: 별도 패널
- 포맷: `format.volume`
- 지표명: `Buy/Sell Volume + Abnormal Volume + Alignment Background`

## 주요 기능

### 1. Buy / Sell Volume 분리

캔들 구조를 바탕으로 거래량을 매수 우위와 매도 우위로 나눠 보여줍니다.

- 종가가 봉 아래보다 위에서 마감할수록 `Buy Volume`이 커집니다.
- 종가가 봉 위쪽보다 아래에서 마감할수록 `Sell Volume` 성격이 강해집니다.

### 2. 비정상 거래량 표시

현재 거래량이 최근 평균보다 충분히 크면 비정상 거래량으로 표시합니다.

주요 설정:

- `비교 캔들 수`
- `비정상 배수`

쉽게 말해, `평소보다 유난히 큰 거래량`이 붙은 봉을 바로 찾는 기능입니다.

### 3. Alignment Background

이번 버전에서 추가된 핵심 기능입니다.

내부적으로 아래 기준을 계산해서 배경색을 보여줍니다.

- 스윙 기준 적응형 VWAP
- Ichimoku Base Line
- 이동평균선(MA)

정렬 조건은 아래와 같습니다.

- 강세 정렬: `MA > Base Line > VWAP`
- 약세 정렬: `MA < Base Line < VWAP`

차트 해석은 아주 단순하게 하면 됩니다.

- 초록 배경: 상승 쪽 정렬이 맞는 구간
- 자주 배경: 하락 쪽 정렬이 맞는 구간

## 추천 사용 흐름

1. 먼저 배경색으로 시장 정렬이 위인지 아래인지 봅니다.
2. 그다음 거래량 막대 색으로 매수/매도 힘을 봅니다.
3. 정렬과 거래량 방향이 같을 때만 신호를 강하게 봅니다.

예시:

- `초록 배경 + 강한 매수 거래량` = 롱 쪽 참고
- `자주 배경 + 강한 매도 거래량` = 숏 쪽 참고
- `배경과 거래량이 반대` = 애매하므로 약하게 보기

## 주의사항

- 이 지표는 자동 진입 신호가 아니라 보조 해석용입니다.
- 거래량 색만 보고 진입하지 말고, 배경 정렬과 가격 구조를 같이 봐야 합니다.
- `비정상 배수`를 너무 낮추면 색이 너무 자주 바뀔 수 있습니다.
- 횡보장에서는 배경색 신호가 자주 바뀌어 해석이 어려워질 수 있습니다.
