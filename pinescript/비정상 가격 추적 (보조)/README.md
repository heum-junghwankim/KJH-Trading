# 비정상 가격 추적 (보조)

트레이딩뷰용 Pine Script 보조지표 설명서입니다.

대상 스크립트:
- [`abnormal-price-tracker-helper.pine`](./abnormal-price-tracker-helper.pine)

이 지표는 `RSI`, `MFI`, `RSI 밴드 폭 상태`를 한 패널에 모아서, 메인 자리 신호 뒤에서 `추격 진입을 줄이고`, `반전 가능성`과 `과열/과매도 상태`를 더 읽기 쉽게 보조하는 용도로 씁니다.

## 이번 업데이트 핵심

- `RSI 선`, `MFI 선`을 각각 켜고 끌 수 있습니다.
- 기본 길이는 `RSI 14`, `MFI 14`입니다.
- 기존 `cRSI` 전용 튜닝값과 과도한 시각화 옵션을 정리했습니다.
- `밴드 폭 축소` 상태의 기본 색상을 `주황색`으로 바꿨습니다.

## 현재 소스 기준으로 보이는 것

| 요소 | 현재 표시 방식 |
| --- | --- |
| RSI 본선 | 기본값 `ON`, 핑크색, 굵기 `2` |
| MFI 본선 | 기본값 `ON`, 청록색, 굵기 `2` |
| 과매수 / 과매도 기준선 | `80 / 20` 회색 점선 |
| MFI 극단 구간 채움 | 과매수는 빨강, 과매도는 초록 채움 |
| RSI 밴드 | 최근 RSI 분포 기준 동적 밴드 |
| 밴드 폭 축소 색상 | 기본값 `주황색` |
| 밴드 폭 확장 색상 | 기본값 `보라색` |
| 우측 상단 패널 | `RSI 밴드` 제목 + 현재 밴드 폭 숫자 |

참고:
- RSI 밴드는 `RSI Length` 기준 값 위에 최근 분포를 다시 읽어 만든 동적 구간입니다.
- 밴드 계산은 기본적으로 `최근 40봉 이상` 구간에서 `10% / 90%` 분위 기준으로 동작합니다.
- 기본 리스크 필터 값은 기존 보조 지표의 `Gap Threshold = 14`, `Strong Gap Threshold = 18`, `Volume Average Length = 20`, `Trend Length = 20`, `보조 최소 강도 = 2` 흐름을 유지합니다.

## 트레이딩 의미 핵심

### 1. 좁은 밴드 색 의미

주황색 계열 밴드는 `RSI 밴드 폭이 좁아진 압축 구간`을 뜻합니다.

- 최근 변동성이 줄고 있다는 뜻입니다.
- 이미 크게 움직인 자리보다 `다음 움직임이 준비되는 구간`에 가깝습니다.
- 이 구간 자체가 바로 진입 신호는 아닙니다.
- 실전에서는 `곧 방향이 나올 수 있으니 메인 신호를 기다리는 준비 구간`으로 보는 편이 좋습니다.

요약하면, 좁은 밴드 색은 `에너지가 접히는 구간`, `추격보다 준비`의 의미가 강합니다.

### 2. 넓은 밴드 색 의미

보라색 계열 밴드 색은 `RSI 밴드 폭이 넓어진 확장 구간`을 뜻합니다.

- 최근 변동성이 이미 많이 벌어졌다는 뜻입니다.
- 추세가 강하게 나온 뒤거나, 위아래 흔들림이 커진 구간일 가능성이 높습니다.
- 새로 추격해서 들어가기엔 리스크가 커질 수 있습니다.
- 실전에서는 `익절 관리`, `트레일링`, `무리한 후행 진입 자제` 쪽에 더 무게를 둡니다.

### 3. MFI 과매도 + RSI가 더 높게 버틸 때 의미

이 상황은 `MFI는 과매도인데 RSI는 그만큼 같이 무너지지 않은 상태`를 뜻합니다.

- 거래량 기준으로는 투매 압력이 강하게 들어왔는데,
- 가격 모멘텀은 상대적으로 덜 꺾였다는 뜻입니다.
- 즉 `매도 압력은 큰데 내부 모멘텀은 덜 무너진 괴리`로 읽을 수 있습니다.

실전에서는 `하락 공포가 과장되었을 수 있는 자리`, `반등 후보를 감시할 자리`로 보는 편이 맞습니다.

### 4. RSI 과열 + MFI가 덜 따라올 때 의미

이 상황은 `RSI가 더 위에 있는데 MFI가 그만큼 따라오지 못한 상태`를 뜻합니다.

- 가격 모멘텀 기준으로는 상승 속도가 과열되어 있는데,
- 자금 유입과 거래량 기준으로는 그 열기를 충분히 확인해주지 못한 상태입니다.
- 즉 `가격은 뜨는데 돈의 추종은 약한 괴리`로 읽을 수 있습니다.

실전에서는 `상승 말단 피로`, `후행 롱 추격 자제`, `익절 관리 우선` 쪽 의미가 큽니다.

## 내부 로직

차트에 보이는 것과 별개로, 내부에서는 반전 위험 점수를 계속 계산합니다.

- `RSI - MFI` 괴리 크기
- 가격 연장 정도
- 거래량 평균 대비 확장 여부
- RSI 극단 구간 또는 RSI 밴드 극단 진입 여부
- 괴리 둔화 여부
- RSI / MFI 방향 전환 여부

이 점수는 최종적으로 아래 알림 조건에 연결됩니다.

- `Bearish Reversal Risk`
- `Bullish Reversal Risk`

즉 이 지표는 `단독 진입 트리거`보다 `지금 추격해도 되는지 다시 거르는 필터`에 더 가깝습니다.

## 자주 만지는 설정

| 설정 | 용도 |
| --- | --- |
| `RSI Length`, `MFI Length` | RSI / MFI 반응 속도 조절 |
| `RSI 선 표시`, `MFI 선 표시` | 두 선을 각각 켜고 끌 때. `MFI`를 끄면 과매수/과매도 채움도 함께 꺼집니다. |
| `RSI Color`, `MFI Color` | 각 선 색상 조절 |
| `기본 밴드 색상` | 일반 상태 RSI 밴드 색상 |
| `좁은 밴드 색상`, `좁은 밴드 폭 기준` | 압축 구간 색과 기준 폭 조절 |
| `넓은 밴드 색상`, `넓은 밴드 폭 기준` | 확장 구간 색과 기준 폭 조절 |
| `Gap Background Threshold` | RSI와 MFI 이격을 눈에 띄게 볼 최소 기준 |
| `Weak Gap Threshold`, `Strong Gap Threshold` | 내부 위험 점수 강도 조절 |
| `Bearish Risk RSI Zone`, `Bullish Risk RSI Zone` | RSI 극단 구간 판정 기준 |
| `Volume Average Length`, `Volume Multiplier` | 거래량 필터 강도 조절 |
| `Trend Length`, `Price Stretch (%)` | 가격 연장 판정 기준 |
| `Use Gap Fade Condition`, `Gap Fade Lookback` | 이격 둔화 확인 속도 조절 |
| `보조 최소 강도` | 약한 단독 괴리보다 겹치는 자리만 더 보수적으로 보려 할 때 |

## 제거된 옵션 방향

이번 정리에서 아래 성격의 옵션은 기본값 고정으로 단순화했습니다.

- `cRSI Dominant Cycle`, `cRSI Vibration`, `cRSI Leveling %`
- 선 굵기 옵션
- 밴드 채움 투명도 세부 옵션

즉 `자주 안 건드리는 세부 튜닝`보다 `실제로 많이 보는 선 표시와 길이`에 집중한 구조입니다.

## 주의사항

- 이 지표는 `방향 확정기`보다 `상태 판독기`에 가깝습니다.
- 주황색 압축 밴드만 보고 바로 진입하기보다, `가격 구조`, `메인 신호`, `거래량`을 같이 보는 편이 안전합니다.
- MFI 극단값과 RSI 이격은 좋은 힌트이지만, 단독으로는 너무 이른 진입 근거가 될 수 있습니다.
- 우측 상단 `RSI 밴드` 숫자는 현재 밴드 폭만 보여주므로, 숫자 자체보다 `색 변화`를 함께 읽는 편이 더 실전적입니다.
