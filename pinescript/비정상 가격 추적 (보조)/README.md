# 비정상 가격 추적 (보조)

트레이딩뷰용 Pine Script 보조지표 설명서입니다.

대상 스크립트:
- [`abnormal-price-tracker-helper.pine`](./abnormal-price-tracker-helper.pine)

이 지표는 `MFI`, `cRSI 밴드`, `cRSI 본선`, `밴드 폭 상태`를 한 패널에 모아서, 메인 자리 신호 뒤에서 `추격 진입을 줄이고`, `반전 가능성`과 `과열/과매도 상태`를 더 읽기 쉽게 보조하는 용도로 씁니다.

## 현재 소스 기준으로 보이는 것

| 요소 | 현재 표시 방식 |
| --- | --- |
| cRSI 본선 | 기본값 `ON`, 핑크색, 굵기 `2` |
| MFI | 청록색, 굵기 `2` |
| 과매수 / 과매도 기준선 | `80 / 20` 회색 점선 |
| MFI 극단 구간 채움 | 과매수는 빨강, 과매도는 초록 채움 |
| cRSI 밴드 | 기본은 은회색, 상태에 따라 밴드선과 밴드 내부 채움 색이 함께 변함 |
| 밴드 폭 축소 색상 | `cRSI Band Range <= cRSI Narrow Band Width Threshold` 일 때 좁은 밴드 색상 적용 |
| 밴드 폭 확장 색상 | `cRSI Band Range >= cRSI Wide Band Width Threshold` 일 때 넓은 밴드 색상 적용 |
| 우측 상단 패널 | `RSI 밴드` 제목 + 현재 밴드 폭 숫자 |
| 우측 상단 숫자 색 | 좁을 때 녹색, 넓을 때 넓은 밴드 색상(기본 보라), 평소엔 흰색 |

참고:
- 패널 이름은 `RSI 밴드`로 보이지만, 내부 계산은 일반 RSI가 아니라 `cRSI 밴드 폭(ub - db)`입니다.
- 밴드 상태 색상은 설정에서 바꿀 수 있습니다. 아래 해석에서는 사용자가 보기 쉽게 `좁은 밴드 색 = 녹색`, `넓은 밴드 색 = 보라색`이라고 부르겠습니다.

## 트레이딩 의미 핵심 4가지

### 1. 좁은 밴드 색 의미

녹색 계열 밴드 색은 `cRSI 밴드 폭이 좁아진 압축 구간`을 뜻합니다.

- 최근 변동성이 줄고 있다는 뜻입니다.
- 이미 크게 움직인 자리보다 `다음 움직임이 준비되는 구간`에 가깝습니다.
- 이 구간 자체가 바로 진입 신호는 아닙니다.
- 실전에서는 `곧 방향이 나올 수 있으니 메인 신호나 MFI 극단값을 기다리는 준비 구간`으로 보는 편이 좋습니다.

요약하면, 좁은 밴드 색은 `에너지가 접히는 구간`, `추격보다 준비`의 의미가 강합니다.

### 2. 넓은 밴드 색 의미

보라색 계열 밴드 색은 `cRSI 밴드 폭이 넓어진 확장 구간`을 뜻합니다.

- 최근 변동성이 이미 많이 벌어졌다는 뜻입니다.
- 추세가 강하게 나온 뒤거나, 위아래 흔들림이 커진 구간일 가능성이 높습니다.
- 새로 추격해서 들어가기엔 리스크가 커질 수 있습니다.
- 실전에서는 `익절 관리`, `트레일링`, `무리한 후행 진입 자제` 쪽에 더 무게를 둡니다.

요약하면, 넓은 밴드 색은 `이미 많이 벌어진 구간`, `공격보다 관리`의 의미가 강합니다.

### 3. MFI 과매도 + RSI 선이 더 높게 이격될 때 의미

이 상황은 `MFI는 과매도인데, cRSI는 그만큼 같이 무너지지 않은 상태`를 뜻합니다.

- 거래량 기준으로는 투매 압력이 강하게 들어왔는데,
- cRSI 기준으로는 모멘텀이 완전히 같은 강도로 눌리지 않았다는 뜻입니다.
- 즉 `매도 압력은 큰데 내부 모멘텀은 덜 무너진 괴리`로 읽을 수 있습니다.

실전 해석:

- 무조건 역매수하는 신호는 아닙니다.
- 다만 `하락 공포가 과장되었을 수 있는 자리`, `반등 후보를 감시할 자리`로는 볼 수 있습니다.
- 특히 가격이 저점을 갱신하지 못하거나, cRSI가 먼저 꺾여 올라오면 `숏 추격 중단` 쪽 의미가 강해집니다.

한 줄로 요약하면, 이 조합은 `투매는 센데 내부 모멘텀은 덜 깨진 상태`라서 `바닥 확인용 관찰 구간`으로 읽는 편이 맞습니다.

### 4. RSI 과매수 + MFI 선이 더 낮게 이격될 때 의미

이 상황은 `cRSI가 과매수권까지 올라갔는데, MFI는 그보다 아래에서 따라오지 못한 상태`를 뜻합니다.

- 가격 모멘텀 기준으로는 상승 속도가 과열되어 있는데,
- 자금 유입과 거래량 기준으로는 그 열기를 충분히 확인해주지 못한 상태입니다.
- 즉 `가격은 뜨는데 돈의 추종은 약한 괴리`로 읽을 수 있습니다.

실전 해석:

- 무조건 역매도하는 신호는 아닙니다.
- 다만 `상승 말단에서 힘이 비어갈 수 있는 자리`, `후행 롱 추격보다 익절과 속도 조절을 우선할 자리`로는 의미가 큽니다.
- 특히 cRSI가 과매수권에서 꺾이거나, MFI가 더 아래에 머물면서 반등하지 못하면 `상단 피로`, `매수 다이버전스 약화`, `숏 감시` 쪽 의미가 강해집니다.

한 줄로 요약하면, 이 조합은 `가격은 과열인데 자금 추종이 약한 상태`라서 `상단 피로 확인 구간`으로 읽는 편이 맞습니다.

## 내부 로직

차트에 보이는 것과 별개로, 내부에서는 반전 위험 점수를 계속 계산합니다.

- `cRSI - MFI` 괴리 크기
- 가격 연장 정도
- 거래량 평균 대비 확장 여부
- cRSI 극단 구간 진입 여부
- 괴리 둔화 여부
- cRSI / MFI 방향 전환 여부

이 점수는 최종적으로 아래 알림 조건에 연결됩니다.

- `Bearish Reversal Risk`
- `Bullish Reversal Risk`

즉 이 지표는 `단독 진입 트리거`보다 `지금 추격해도 되는지 다시 거르는 필터`에 더 가깝습니다.

## 자주 만지는 설정

| 설정 | 용도 |
| --- | --- |
| `Show cRSI Line` | cRSI 본선 표시 여부 |
| `cRSI Color`, `MFI Color` | cRSI / MFI 선 색상 |
| `cRSI Line Width`, `MFI Line Width` | 선 굵기 |
| `cRSI Band Color`, `cRSI Band Width` | cRSI 밴드 외곽선 스타일 |
| `cRSI Band Fill Color`, `cRSI Band Fill Transparency` | cRSI 밴드 채움 스타일 |
| `cRSI Narrow Band Color`, `cRSI Narrow Band Fill Transparency` | 좁은 밴드 구간 선/채움 스타일 |
| `cRSI Narrow Band Width Threshold` | 이 값 이하이면 좁은 밴드 상태로 판단 |
| `cRSI Wide Band Color`, `cRSI Wide Band Fill Transparency` | 넓은 밴드 구간 선/채움 스타일 |
| `cRSI Wide Band Width Threshold` | 이 값 이상이면 넓은 밴드 상태로 판단 |
| `MFI Length` | MFI 반응 속도 조절 |
| `Gap Threshold` | cRSI와 MFI 이격을 눈에 띄게 볼 최소 기준 |
| `Weak Gap Threshold`, `Strong Gap Threshold` | 내부 위험 점수 강도 조절 |
| `Bearish Risk cRSI Zone`, `Bullish Risk cRSI Zone` | cRSI 극단 구간 판정 기준 |
| `Volume Average Length`, `Volume Multiplier` | 거래량 필터 강도 조절 |
| `Trend Length`, `Price Stretch (%)` | 가격 연장 판정 기준 |
| `Use Gap Fade Condition`, `Gap Fade Lookback` | 이격 둔화 조건 조절 |

## 주의사항

- 이 지표는 `방향 확정기`보다 `상태 판독기`에 가깝습니다.
- 녹색 배경이나 보라색 배경만 보고 바로 진입하기보다, `가격 구조`, `메인 신호`, `거래량`을 같이 보는 편이 안전합니다.
- MFI 극단값과 cRSI 이격은 좋은 힌트이지만, 단독으로는 너무 이른 진입 근거가 될 수 있습니다.
- 우측 상단 `RSI 밴드` 숫자는 현재 밴드 폭만 보여주므로, 숫자 자체보다 `색 변화`와 `배경 상태`를 함께 읽는 편이 더 실전적입니다.
