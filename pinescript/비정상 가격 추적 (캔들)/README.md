# 비정상 가격 추적 (캔들)

트레이딩뷰용 Pine Script 오버레이 지표 설명서입니다.

대상 스크립트:
- [`abnormal-price-tracker-candles.pine`](./abnormal-price-tracker-candles.pine)

문서 생성 스크립트:
- [`generate_candle_guide_pdf.py`](./generate_candle_guide_pdf.py): `FPDF`로 초보자용 시각 가이드 PDF를 생성합니다.
- [`generate_candle_guide_weasy.py`](./generate_candle_guide_weasy.py): `WeasyPrint`로 같은 가이드를 HTML 기반 PDF로 생성합니다.

이 지표는 `자리 -> 다이버전스 -> ATR 감시` 흐름을 한 장에서 읽기 위한 도구입니다. `0단계 비정상 캔들`과 `1단계 확정 다이버전스`는 봉 마감 기준으로 확정되고, `1단계 프리뷰`와 `2단계 ATR 배경 / ATR 강화`는 현재 진행 중인 봉에서도 실시간으로 반응할 수 있습니다.

## 현재 소스 기준으로 보이는 것

| 요소 | 현재 표시 방식 |
| --- | --- |
| 0단계 비정상 캔들 | 과매수는 주황 계열, 과매도는 하늘 계열로 봉 색을 바꿉니다. |
| 1단계 다이버전스 | 프리뷰는 실시간 원형으로, 확정 신호는 봉 마감 직후 삼각형으로 표시합니다. |
| 2단계 ATR 배경 / ATR 강화 | 자리 이후 실제 후속 이동이 붙으면 배경이 켜지고, 더 강하면 강화 색으로 바뀌며 현재 진행 중인 봉에서도 바로 반영될 수 있습니다. |

## 트레이딩 의미 핵심

### 1. 비정상 캔들 의미

비정상 캔들은 `반전 확정`이 아니라 `자리 경고`입니다.

- cRSI 극단과 볼린저 몸통 이탈이 겹친 봉만 먼저 잡습니다.
- 즉 `지금 바로 뒤집힌다`보다 `여기서부터 반응을 감시할 만하다`에 가깝습니다.

요약하면, 0단계는 `자리 후보 시작`입니다.

### 2. 다이버전스 의미

다이버전스는 `강세/약세 효율 저하 힌트`입니다.

- 피벗 확정 뒤에 나오는 신호라서 무작정 빠른 신호는 줄여 줍니다.
- 자리 후보 뒤에 붙으면 `반전 가능성 감시` 의미가 커집니다.

요약하면, 1단계는 `힌트`이고 아직 실행 자체를 뜻하지는 않습니다.

### 3. ATR 배경과 ATR 강화 의미

ATR 배경은 `자리 이후 실제 후속 이동이 붙었는지`, ATR 강화는 `그 후속 이동이 더 강한지`를 뜻합니다.

- 다이버전스나 비정상 캔들만으로 끝나지 않고 실제 움직임이 이어지는지 확인하는 단계입니다.
- 감시가 본격적으로 살아나는 구간은 보통 ATR 배경부터입니다.

요약하면, 2단계는 `감시 시작과 감시 강화`입니다.

## 내부 로직

### 1. 비정상 캔들 판정

비정상 캔들은 아래 두 조건이 겹칠 때만 확정됩니다.

1. cRSI가 절대 과매수/과매도 구간에 들어옴
2. 캔들 몸통이 볼린저 밴드를 `비정상 캔들 몸통 밴드 초과 비율` 이상 벗어남

봉 색상은 별도 시각 조건으로 한 번 더 거릅니다.

- 윗꼬리/아랫꼬리까지 포함한 시각적 이탈 비율이 `비정상 캔들 시각 표시 밴드 초과 비율` 이상일 때만 실제 색을 칠합니다.
- 그래서 내부적으로 비정상 조건이 성립해도, 시각 이탈이 약하면 봉 색상은 기본값으로 남을 수 있습니다.

### 2. ATR 배경 판정

배경은 `다이버전스 기준`과 `비정상 캔들 기준` 두 갈래로 계산합니다.

- 기준 신호 이후 추적 봉 수 안에서만 유효합니다.
- 기준 봉의 참조 가격 대비 실제 가격 이동이 `ATR 과도 이동 배수` 또는 `비정상 캔들 기준 ATR 과도 이동 배수` 이상이어야 합니다.
- RSI 이동이 `RSI 추가 이동 제한값`을 넘지 않아야 합니다.
- 옵션에 따라 cRSI 필터를 추가로 통과해야 합니다.
- 위 조건을 넘는 순간, 봉 마감 전이라도 현재 봉 배경이 먼저 켜질 수 있습니다.

강한 배경은 아래 중 하나면 성립합니다.

- 다이버전스 기준 배경과 비정상 캔들 기준 배경이 동시에 켜짐
- 각 기준에서 가격 이동이 `강한 배경 판정 ATR 배수` 또는 `비정상 캔들 기준 강한 배경 ATR 배수` 이상으로 확장됨

### 3. 알림

현재 스크립트에는 아래 알림 흐름이 들어 있습니다.

- `0단계`: 비정상 과매수/과매도 시작, 비정상 상태 유지
- `1단계`: 강세/약세 다이버전스 프리뷰, 봉 마감 직후 강세/약세 다이버전스 확정
- `2단계`: ATR 배경 진입, ATR 배경 강화 (봉중 실시간 반응 가능)

## 자주 만지는 설정

| 설정 | 언제 조정하나 |
| --- | --- |
| `비정상 캔들 볼린저 기간`, `비정상 캔들 볼린저 배수` | 자리 판정 민감도를 바꾸고 싶을 때 |
| `비정상 캔들 몸통 밴드 초과 비율` | 몸통이 밴드 밖으로 얼마나 확실히 나와야 비정상으로 볼지 조절할 때 |
| `비정상 캔들 시각 표시 밴드 초과 비율` | 실제 봉 색상이 너무 자주 또는 너무 적게 바뀔 때 |
| `배경 구간 표시` | ATR 감시 배경 자체를 켜고 끌 때 |
| `다이버전스 기준 배경 사용`, `비정상 과매수/과매도 캔들 기준 배경 사용` | 어떤 종류의 자리 신호를 배경 추적에 반영할지 고를 때 |
| `기준 신호 이후 추적 봉 수`, `비정상 캔들 기준 추적 봉 수` | 자리 이후 감시를 얼마나 오래 유지할지 바꿀 때 |
| `ATR 과도 이동 배수`, `비정상 캔들 기준 ATR 과도 이동 배수` | ATR 배경이 더 빠르게 또는 더 늦게 켜지게 하고 싶을 때 |
| `강한 배경 판정 ATR 배수`, `비정상 캔들 기준 강한 배경 ATR 배수` | 강화 배경 기준을 더 엄격하게 또는 더 느슨하게 할 때 |
| `배경 구간에 cRSI 필터 적용` | 약한 후속 움직임을 더 많이 걸러내고 싶을 때 |

## 주의사항

- 자동매매 전략이 아니라 해석 보조 지표입니다.
- `0단계 비정상 캔들`과 `1단계 확정 다이버전스`는 봉 마감 기준입니다.
- `2단계 ATR 배경 / ATR 강화`는 기준 신호가 생긴 뒤 현재 봉 중에도 먼저 반응할 수 있습니다.
- 다이버전스는 피벗 확정 뒤에 표시되므로 지연이 있습니다.
