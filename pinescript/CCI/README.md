# CCI 익스트림 크로스 시그널

트레이딩뷰용 Pine Script 보조 지표 설명서입니다.

대상 스크립트:
- [`cci-extreme-cross-signals.pine`](./cci-extreme-cross-signals.pine)

이 지표는 기본 `CCI`에 `CCI 기준 이동평균`을 겹쳐서, 극단값 구간을 먼저 찍은 뒤 다시 평균선을 재돌파하는 순간만 시그널로 보여주는 버전입니다. 단순히 `CCI가 0선을 넘었는가`보다, `과매도/과매수 뒤 재진입`에 초점을 둡니다.

## 현재 소스 기준으로 보이는 것

| 요소 | 현재 표시 방식 |
| --- | --- |
| CCI Oscillator | 현재 가격이 평균에서 얼마나 벗어났는지 보는 진동선입니다. |
| CCI-based MA | 기본값 `SMA`, 길이 `14`인 CCI 기준 신호선입니다. |
| Bullish / Bearish Extreme | 기본값 `-175 / 175`로 먼저 극단 구간 진입 여부를 확인합니다. |
| Upper / Lower Band | 현재 기본값도 `175 / -175`로, 극단 구간 시각 기준선 역할을 합니다. |
| Re-entry Signal | 극단값 진입 뒤 CCI가 신호선을 다시 넘을 때만 점 형태로 표시됩니다. |
| Signal Offset | 기본값 `0`, 신호 확정 시점을 바꾸지 않고 차트 표시 위치만 이동합니다. |

참고:
- 현재 기본값은 `CCI Length = 20`, `CCI-based MA Type = SMA`, `CCI-based MA Length = 14`입니다.
- `Bullish Extreme = -175`, `Bearish Extreme = 175`가 먼저 찍혀야 재진입 점 시그널이 활성화됩니다.

## 트레이딩 의미 핵심 4가지

### 1. 과매도 Extreme 아래 진입 의미

`Bullish Extreme` 아래로 내려간 상태는 `평균 대비 하락 압력이 과도하게 벌어진 구간`을 뜻합니다.

- 단순히 싸다는 뜻보다는 `한쪽으로 과하게 쏠린 상태`에 가깝습니다.
- 이 단계만으로는 바로 롱 신호가 아니라, `되돌림 준비 여부를 감시할 자리`로 보는 편이 맞습니다.

요약하면, 이 구간은 `즉시 진입`보다 `극단값 확인`의 의미가 더 큽니다.

### 2. 과매도 뒤 녹색 재진입 점 의미

과매도 Extreme을 먼저 찍은 뒤 녹색 점이 나오면 `과하게 눌린 모멘텀이 다시 신호선 위로 복귀하는 첫 확인`에 가깝습니다.

- 급락 뒤 반등 시도를 빠르게 포착하는 데 유용합니다.
- 메인 자리 신호와 겹치면 `추격 매도 중단`, `반등 확인` 쪽 의미가 커집니다.

요약하면, 녹색 점은 `과매도 이후 재진입 확인`입니다.

### 3. 과매수 뒤 빨간 재진입 점 의미

과매수 Extreme을 먼저 찍은 뒤 빨간 점이 나오면 `과하게 올라간 모멘텀이 다시 아래로 꺾이는 첫 확인`에 가깝습니다.

- 급등 뒤 상승 피로를 보는 데 유용합니다.
- 메인 신호가 약해지는 자리와 겹치면 `후행 롱 자제`, `숏 감시` 의미가 강해집니다.

요약하면, 빨간 점은 `과매수 이후 하락 재개 확인`입니다.

### 4. 단순 크로스만 있고 점이 없을 때 의미

CCI와 신호선이 교차해도 점이 없으면 `극단값 필터를 통과하지 못한 일반 잡음성 크로스`로 보는 편이 좋습니다.

- 방향 힌트는 줄 수 있지만, 강한 재진입 신호로 보기는 어렵습니다.
- 실전에서는 이 경우 `다른 지표 확인 전까지는 보류` 쪽이 더 안전합니다.

## 내부 로직

이 지표는 `극단값 진입 -> 신호선 재돌파` 두 단계를 모두 통과해야 점 시그널을 만듭니다.

- 먼저 `CCI Length` 기준으로 CCI를 계산합니다.
- 그다음 CCI 위에 `CCI-based MA`를 얹어 재진입 기준선을 만듭니다.
- 강세 시그널은 `Bullish Extreme` 아래 진입 이력이 있는 상태에서 CCI가 신호선을 다시 위로 넘을 때만 나옵니다.
- 약세 시그널은 `Bearish Extreme` 위 진입 이력이 있는 상태에서 CCI가 신호선을 다시 아래로 깰 때만 나옵니다.

즉 단순 크로스 지표가 아니라, `먼저 과열/과매도 구간이 있었는지`를 전제로 한 `재진입 확인기`에 가깝습니다.

추가로 알아둘 점:

- 이동평균 타입을 `None`으로 두면 신호선이 없어지므로 재진입 점 시그널도 함께 비활성화됩니다.
- `Signal Offset`은 계산 로직을 바꾸지 않고, 차트에서 보이는 위치만 시각적으로 이동합니다.

## 자주 만지는 설정

| 설정 | 언제 조정하나 |
| --- | --- |
| `CCI Length` | CCI 민감도를 더 빠르게 또는 느리게 보고 싶을 때 |
| `CCI-based MA Type`, `CCI-based MA Length` | CCI 위에 얹는 신호선 성격을 바꾸고 싶을 때 |
| `Bullish Extreme`, `Bearish Extreme` | 극단값 기준을 더 엄격하게 또는 느슨하게 바꿀 때 |
| `Upper Band`, `Lower Band` | 화면 기준선을 극단값 기준과 다르게 보고 싶을 때 |
| `Signal Offset` | 확정봉 그대로 볼지, 시각적으로 한 봉 이동해 볼지 정리할 때 |

## 주의사항

- `Signal Offset = 1`은 신호가 확정되는 시점을 바꾸는 것이 아니라, 차트에서 표시만 한 봉 앞당겨 보이게 합니다.
- 이동평균을 `None`으로 두면 극단값 및 재진입 시그널도 함께 비활성화됩니다.
- 단독 진입 근거보다, 자리나 거래량 지표와 함께 보는 보조 확인용에 가깝습니다.
