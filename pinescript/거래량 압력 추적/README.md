# 거래량 압력 추적 / 볼륨 델타

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine)
- [`volume-delta.pine`](../볼륨%20델타/volume-delta.pine)
- [`볼륨 델타 README`](../볼륨%20델타/README.md)

## 개요

이 폴더의 거래량 관련 지표는 역할을 두 개로 나눴습니다.

- `거래량 압력 추적`: 현재 봉의 `Buy Volume / Sell Volume` 압력과 비정상 거래량, 전체 주문량 평균선을 봅니다.
- `볼륨 델타`: 차트 위에 직접 표시되며, 사용자가 정한 집계 구간 안에서 `매수/매도 총량`, `비율`, `델타`, `가격 변화율`을 요약해서 봅니다.

즉:

- `거래량 압력 추적`은 `지금 봉에서 어느 쪽 압력이 강한가`
- `볼륨 델타`는 `같은 구간에서 누가 더 우세했고 가격이 실제로 얼마나 움직였는가`

## 트레이딩뷰 적용 방법

### 거래량 압력 추적

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.

### 볼륨 델타

1. 새 Pine Editor 탭을 엽니다.
2. [`volume-delta.pine`](../볼륨%20델타/volume-delta.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.

세부 설명은 [`볼륨 델타 README`](../볼륨%20델타/README.md)에서 따로 볼 수 있습니다.

## 거래량 압력 추적

![지표 예시 화면](./img.png)

이 스크립트는 아래 3가지를 보여줍니다.

- 흰색/회색 막대: 일반 매수/매도 거래량
- 연두색/빨간색 막대: 비정상 거래량이 붙은 매수/매도 압력 구간
- 주황색 선: 전체 주문량 평균선

핵심 기능:

| 기능 | 핵심 의미 | 주요 설정 |
| --- | --- | --- |
| Buy / Sell Volume 분리 | 종가 위치 기준으로 현재 봉의 매수/매도 압력을 나눕니다. | `정상 Buy 색상`, `정상 Sell 색상` |
| 비정상 거래량 표시 | 최근 평균보다 큰 봉만 바로 눈에 띄게 강조합니다. | `비교 캔들 수`, `비정상 배수` |
| 전체 주문량 평균선 | 최근 전체 주문량 평균 흐름을 같이 봅니다. | `평균선 표시`, `평균선 봉 수` |

짧게 보면:

- `막대 색`은 압력 방향
- `비정상 색상`은 평소보다 강한 체결
- `평균선 상회`는 실제 힘이 붙는지 보는 기준

## 볼륨 델타

이 스크립트는 차트 위에 요약 박스로 아래 값을 보여줍니다.

- `총합`
- `매수 총량`과 `비율`
- `매도 총량`과 `비율`
- `델타`
- `가격 변화율`

핵심 기능:

| 기능 | 핵심 의미 | 주요 설정 |
| --- | --- | --- |
| 집계 기준 선택 | `세션` 또는 `특정 봉 수` 기준으로 집계합니다. | `집계 기준`, `봉 수` |
| 볼륨 델타 요약 | 같은 구간의 매수/매도 우세를 숫자로 확인합니다. | `볼륨 델타 요약 표시`, `표시 위치` |
| 배경 투명도 조절 | 헤더와 일반 필드 배경에 같은 투명도를 적용합니다. | `배경 투명도` |
| 가격 변화율 비교 | 델타 방향이 실제 가격 변화로 이어졌는지 봅니다. | 집계 구간과 동일한 기준 사용 |

해석 기준은 단순합니다.

- `델타 플러스` + `가격 변화율 플러스` = 매수 우세가 실제 상승으로 이어짐
- `델타 마이너스` + `가격 변화율 마이너스` = 매도 우세가 실제 하락으로 이어짐
- `델타`와 `가격 변화율` 방향이 어긋남 = 흡수, 분산, 반전 테스트 가능성

## 같이 쓰는 방법

1. `거래량 압력 추적`으로 현재 봉에 비정상 매수/매도 압력이 붙는지 먼저 봅니다.
2. 같은 자리에서 `전체 주문량 평균선` 위로 거래량이 실리는지 확인합니다.
3. `볼륨 델타`에서 같은 구간의 `매수/매도 비율`, `델타`, `가격 변화율`이 어느 쪽으로 정렬되는지 봅니다.

예시:

- 롱: 연두색 비정상 매수 막대 + 전체 평균선 상회 + 델타 플러스 + 가격 변화율 플러스
- 숏: 빨간 비정상 매도 막대 + 전체 평균선 상회 + 델타 마이너스 + 가격 변화율 마이너스
- 관망: 막대는 강한데 델타와 가격 변화율이 서로 어긋날 때

## 주의사항

- 두 스크립트 모두 실제 호가 체결 델타가 아니라 `봉 내부 종가 위치 기반` 추정치입니다.
- 그래서 자동 진입 신호보다는 `압력 확인`과 `흡수/불균형 해석` 용도로 쓰는 편이 좋습니다.
- 지표 하나만 보고 진입하지 말고, 추세와 가격 구조를 같이 확인해야 합니다.
