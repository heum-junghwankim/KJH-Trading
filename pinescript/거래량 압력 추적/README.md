# 거래량 압력 추적

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine)

## 개요

이 지표는 거래량을 `Buy Volume`과 `Sell Volume`으로 나눠서, 현재 시장의 `거래량 압력`이 어느 쪽에 더 실리는지 보여줍니다.

여기에 `비정상 거래량`과 `추세 정렬 배경`까지 함께 표시해서, 단순 거래량 크기보다 `실제 매수/매도 압력`을 더 해석하기 쉽게 만든 지표입니다.

## 사용 지표

이 스크립트는 아래 지표를 조합해서 거래량 압력과 방향 정렬을 봅니다.

- `Buy Volume / Sell Volume`
- `Abnormal Volume`
- `Dynamic Swing Anchored VWAP`
- `Ichimoku Base Line`
- `Simple Moving Average`

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.
5. 필요하면 저장합니다.

## 예시 화면

![지표 예시 화면](./img.png)

위 이미지에서 볼 수 있는 핵심은 아래 3가지입니다.

- 흰색/회색 막대: 일반 매수/매도 거래량
- 연두색/빨간색 막대: 비정상 거래량이 붙은 매수/매도 압력 구간
- 진한 초록/연한 초록, 진한 자주/연한 자주 배경: 추세 정렬이 맞는 구간과 강도 차이

쉽게 해석하면:

- `막대 색`은 거래량 압력의 성격
- `배경 색`은 시장 방향의 정렬 상태

## 이미지에서 어떻게 보는지

이미지에서 이 지표는 `마지막 거래량 압력 확인`에 가깝습니다.

- 초록 배경이면 상승 정렬, 자주 배경이면 하락 정렬입니다.
- 배경이 더 진할수록 정렬 강도가 더 강한 구간으로 해석합니다.
- 연두색 비정상 거래량은 `평소보다 강한 매수 유입`으로 봅니다.
- 빨간 비정상 거래량은 `평소보다 강한 매도 압력`으로 봅니다.
- 횡보나 전환 구간에서는 초록/자주 배경이 짧게 교차해서 나타날 수 있습니다.

핵심은 거래량 막대 하나보다 `배경 방향과 같은 쪽의 압력이 비정상적으로 커지는지`입니다.

- 초록 배경 + 연두 막대 = 롱 쪽 힘 확인
- 자주 배경 + 빨간 막대 = 숏 쪽 힘 확인
- 배경과 막대가 반대 = 추격보다 관망 우선

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: 별도 패널
- 포맷: `format.volume`
- 지표명: `Buy/Sell Volume + Abnormal Volume + Alignment Background`

## 주요 기능

| 기능 | 핵심 의미 | 주요 설정/조건 |
| --- | --- | --- |
| Buy / Sell Volume 분리 | 종가 위치를 기준으로 매수/매도 압력을 나눠 보여줍니다. | 종가가 위쪽이면 `Buy`, 아래쪽이면 `Sell` 성격 강화 |
| 비정상 거래량 표시 | 최근 평균보다 큰 거래량만 바로 눈에 띄게 표시합니다. | `비교 캔들 수`, `비정상 배수` |
| Alignment Background | 추세 정렬이 위인지 아래인지 배경색으로 보여줍니다. | 강세: `MA > Base > VWAP`, 약세: `MA < Base < VWAP` |
| 정렬 강도 표현 | 배경 진하기로 정렬 간격의 강도를 보여줍니다. | `MA-Base`, `Base-VWAP` 중 더 약한 쪽 기준 |

짧게 보면:

- 막대 색은 `압력 방향`
- 연두/빨강은 `비정상 체결`
- 초록/자주는 `정렬 방향`
- 배경 진하기는 `정렬 강도`

기본 추세 판단 기준은 `MA 5 / Base 20 / Swing 20`을 먼저 보는 쪽이 단순합니다.

## 추천 사용 흐름

1. 먼저 배경색으로 시장 정렬이 위인지 아래인지 봅니다.
2. 그다음 거래량 막대 색으로 매수/매도 압력을 봅니다.
3. 정렬과 거래량 압력 방향이 같을 때만 신호를 강하게 봅니다.

예시:

- `초록 배경 + 강한 매수 압력` = 롱 쪽 참고
- `자주 배경 + 강한 매도 압력` = 숏 쪽 참고
- `배경과 거래량 압력이 반대` = 애매하므로 약하게 보기

## 거래 예시

### 롱 예시

- 추세 지표가 `HH/HL` 구조이고 초록 정렬이 유지됩니다.
- 캔들 지표에서 눌림 이후 강세 신호가 나옵니다.
- 이 거래량 지표에서 초록 배경 안에 연두색 비정상 매수 압력 막대가 터집니다.
- 그러면 `상승 정렬 안에서 실제 매수 참여가 붙었다`고 해석하고 롱 쪽 확률을 높게 봅니다.

### 숏 예시

- 추세 지표가 `LH/LL` 구조이고 자주 정렬이 유지됩니다.
- 캔들 지표에서 반등 이후 약세 신호가 나옵니다.
- 이 거래량 지표에서 자주 배경 안에 빨간 비정상 매도 압력 막대가 터집니다.
- 그러면 `하락 정렬 안에서 실제 매도 참여가 붙었다`고 해석하고 숏 쪽 확률을 높게 봅니다.

즉, 이 지표는 방향을 처음 정하는 도구가 아니라 `이미 잡아둔 시나리오에 실제 힘이 붙는지 확인하는 도구`로 쓰면 가장 좋습니다.

## 세력 관점 해석

![거래량 압력 추적 세력 관점 다이어그램](./smart-money-view.svg)

이 지표는 `누가 더 크게 체결하고 있는가`보다 `어느 방향으로 큰 체결이 반복되는가`를 보는 데 의미가 있습니다.

### 1. `초록 배경 + 연두색 비정상 매수 압력`

가장 이상적인 강세 확인 구간입니다.

- 정렬이 이미 위를 가리키는데 비정상 매수 압력까지 붙으면, 위로 가격을 밀고 싶은 큰 자금이 실제로 체결까지 붙이고 있다고 해석할 수 있습니다.
- 특히 눌림 뒤 이런 막대가 나오면 `눌림 흡수 후 재가속` 가능성을 높게 봅니다.
- 예상되는 움직임은 `저점 방어 -> 직전 고점 재도전 -> 돌파 시도`입니다.

세력 관점으로는 `유동성 스윕 후 매수 흡수` 또는 `추세 지속용 추가 매집` 시나리오에 가깝습니다.

### 2. `자주 배경 + 빨간 비정상 매도 압력`

가장 이상적인 약세 확인 구간입니다.

- 정렬이 아래를 가리키는데 비정상 매도 압력까지 붙으면, 반등을 이용해 위에서 물량을 던지거나 신규 숏이 붙는 흐름일 수 있습니다.
- 반등 실패 뒤 이 조합이 반복되면 `위로 끌어올렸다가 다시 누르는 공급 우위` 구조로 볼 수 있습니다.
- 예상되는 움직임은 `반등 제한 -> 저점 재시험 -> 하단 이탈 시도`입니다.

세력 관점으로는 `반등 구간 분산` 또는 `하락 추세 재개`를 우선 의심합니다.

### 3. 배경과 거래량 압력이 반대일 때

이때가 가장 중요합니다. 많은 경우 `함정`이 여기서 나옵니다.

- `초록 배경 + 빨간 비정상 매도 압력`
  - 상승 추세 안의 `롱 청산`, `단기 손절 유도`, `눌림 테스트`일 수 있습니다.
  - 다음 봉에서 다시 가격을 회복하고 매수 압력이 붙으면 `흡수` 가능성이 큽니다.
  - 그러나 기준선 아래로 밀리며 빨간 압력이 이어지면 `분산 시작`일 수 있습니다.

- `자주 배경 + 연두 비정상 매수 압력`
  - 하락 추세 안의 `숏 커버`, `단기 반등`, `바닥 훼손 후 되감기`일 수 있습니다.
  - 다음 봉에서 고점을 못 넘고 다시 매도 압력이 붙으면 `반등 함정`으로 끝날 가능성이 높습니다.
  - 반대로 정렬까지 위로 바뀌면 `진짜 바닥 흡수` 가능성을 다시 봅니다.

즉 반대색 비정상 거래량은 `즉시 반전 신호`가 아니라 `누군가 반대 물량을 받아내고 있는지 테스트하는 구간`으로 보는 편이 안전합니다.

### 4. 비정상 거래량이 계속 큰데 가격이 안 가는 경우

이 경우는 `흡수(absorption)` 가능성을 봐야 합니다.

- 매수 비정상 거래량이 반복되는데 윗꼬리만 길고 종가가 못 버티면, 위에서 큰 물량이 계속 받아치고 있을 수 있습니다.
- 매도 비정상 거래량이 반복되는데 저점을 못 깨고 아래꼬리만 길면, 아래에서 큰 자금이 계속 받아내고 있을 수 있습니다.

거래량이 크다고 항상 추세가 이어지는 것은 아니고, `큰 체결이 어디서 소화되는가`가 더 중요합니다.

### 5. 이 지표로 예측할 수 있는 움직임

- 정렬과 같은 방향의 비정상 거래량: 추세 지속 가능성 증가
- 정렬과 반대 방향의 비정상 거래량: 흔들기, 청산, 단기 함정 가능성 증가
- 비정상 거래량 반복 + 가격 정체: 흡수 또는 분산 진행 가능성 증가
- 비정상 거래량 감소 + 배경 약화: 추세 탄력 저하 가능성 증가

## 실전 롱/숏 전략 예시

![거래량 압력 추적 롱숏 전략 예시](./trade-strategy-example.svg)

### 롱 기준

- `롱 매수`: 가격이 눌림 뒤 다시 회복하는 자리에서 `초록 배경 + 연두 비정상 매수`가 같이 붙을 때 진입합니다.
- `롱 매도`: 목표 저항 구간에서 `빨강 비정상 매도`가 반대로 나오면 우선 청산을 생각합니다.
- `무효화`: 초록 배경이 사라지고, 빨간 비정상 매도가 연속으로 붙으면 시나리오를 약화시킵니다.

### 숏 기준

- `숏 매도`: 반등 실패 자리에서 `자주 배경 + 빨강 비정상 매도`가 같이 붙을 때 진입합니다.
- `숏 매수`: 하단 지지 구간에서 `연두 비정상 매수`가 반대로 나오면 우선 청산을 생각합니다.
- `무효화`: 자주 배경이 약해지고, 연두 비정상 매수가 반복되면 숏 시나리오를 줄입니다.

### 이 지표를 쓸 때 중요한 점

- 거래량 지표는 `진입 이유`보다 `진입해도 되는지 확인`하는 용도에 가깝습니다.
- 배경과 같은 방향의 비정상 거래량만 강하게 보고, 반대색은 `함정`부터 의심합니다.
- 진입 후에도 반대색 비정상 거래량이 나오면 신규 진입보다 `청산/보수화`를 먼저 생각합니다.

## 주의사항

- 이 지표는 자동 진입 신호가 아니라 보조 해석용입니다.
- 거래량 색만 보고 진입하지 말고, 배경 정렬과 가격 구조를 같이 봐야 합니다.
- `비정상 배수`를 너무 낮추면 색이 너무 자주 바뀔 수 있습니다.
- 횡보장에서는 배경색 신호가 자주 바뀌어 해석이 어려워질 수 있습니다.
- 이 계산은 실제 호가창의 체결 분류가 아니라 `봉 내부 종가 위치 기반` 압력 해석이므로, 반드시 가격 구조와 함께 봐야 합니다.
