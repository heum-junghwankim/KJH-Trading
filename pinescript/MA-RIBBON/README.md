# 이평선 리본

트레이딩뷰용 Pine Script 오버레이 지표 설명서입니다.

대상 스크립트:
- [`moving-average-ribbon.pine`](./moving-average-ribbon.pine)

이 지표는 `20, 25, 30, 35, 40, 45, 50` 이동평균을 리본처럼 겹쳐 보여 주고(`SMA / EMA / WMA / VWMA` 중 선택), `MA 100 / 200` 장기선과 두 선 사이 배경을 함께 그립니다. 추세 방향·정렬 구조를 한눈에 보는 순수 리본 오버레이입니다.

> 이전에 있던 `횡보 압축 / 상승 분출 / 하락 분출` 표시는 이치모쿠 인디케이터·전략으로 이전되었습니다. 해당 로직은 [ICHIMOKU README](../ICHIMOKU/README.md)를 참고하세요.

## 표시

| 요소 | 표시 방식 |
| --- | --- |
| MA 20~50 | 노랑~주황 그라데이션 7개 라인(리본) |
| MA 100 | 빨강 라인 (`MA 100` 토글) |
| MA 200 | 진한 빨강 라인 (`MA 200` 토글) |
| MA 100/200 사이 배경 | `MA100 > MA200`이면 초록, 아니면 빨강 채움 |

## 설정

- `Source`: 이동평균 계산에 쓰는 가격 소스(기본 `종가`).
- `이동평균 종류`: `SMA`, `EMA`, `WMA`, `VWMA` 중 하나를 선택합니다.
- `MA 100`, `MA 200`: 장기 기준선과 두 선 사이 배경 채우기 표시 여부를 정합니다.

## 해석 팁

- 20~50 리본이 **우상향으로 순차 정렬**되면 중기 상승 구조, 그 반대면 중기 하락 구조로 봅니다.
- 리본이 좁게 **뭉치면 횡보**, 넓게 **벌어지면 추세**가 진행 중일 가능성이 큽니다.
- `MA 100/200` 배경 색은 장기 추세 방향(초록=정배열, 빨강=역배열)을 큰 그림으로 보여 줍니다.
- 압축/분출을 신호로 활용하려면 이치모쿠 인디케이터·전략의 `리본 압축/분출` 다이아몬드 표시를 사용하세요.
