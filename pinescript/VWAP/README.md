# Advanced Multi Anchor VWAP

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`multi-vwap.pine`](./multi-vwap.pine)

## 개요

이 지표는 `Session VWAP`, `현재 일봉 포함 최근 5일 VWAP`, `현재 일봉 포함 최근 20일 VWAP`를 차트 위에 동시에 표시하는 오버레이 스크립트입니다.

핵심은 단순히 `현재가가 Session VWAP 위인가`만 보는 것이 아니라, `오늘 평균 단가`, `최근 5거래일 평균 단가`, `최근 20거래일 평균 단가`가 어떤 순서로 정렬되는지까지 한 번에 보는 것입니다.

여기에 `Session VWAP` 기준 3단계 밴드와 `표준편차 / 퍼센트` 기반 밴드 계산을 유지해서, 기준선은 여러 개로 넓게 보고 밴드는 가장 짧은 기준 하나에 집중해서 해석할 수 있도록 구성했습니다.

## 사용 지표

이 스크립트는 아래 요소를 조합해서 기준 가격대를 봅니다.

- `Session VWAP`
- `5-Day Rolling VWAP` (`현재 일봉 포함 최근 5봉`)
- `20-Day Rolling VWAP` (`현재 일봉 포함 최근 20봉`)
- `Session VWAP` 기준 표준편차 밴드
- `Session VWAP` 기준 퍼센트 밴드

짧게 보면:

- `Session VWAP`는 오늘 평균 단가
- `5-Day VWAP`는 단기 스윙 평균 단가
- `20-Day VWAP`는 더 큰 기준 평균 단가
- `Band`는 `Session VWAP`에서 얼마나 이격됐는지

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`multi-vwap.pine`](./multi-vwap.pine) 파일의 전체 내용을 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `Add to chart` 또는 `차트에 추가`를 클릭합니다.
5. 필요하면 `Save`로 개인 스크립트로 저장합니다.

## 이미지에서 어떻게 보는지

이 지표는 `지금 가격이 어느 기간 평균 위에 있는지`를 한 번에 확인하는 용도입니다.

먼저 보는 순서는 아래처럼 단순하게 가져가면 됩니다.

1. 가격이 `Session VWAP` 위인지 아래인지 먼저 봅니다.
2. 그다음 `Session / 5-Day / 20-Day`가 위아래로 어떤 순서로 정렬되는지 봅니다.
3. 마지막으로 가격이 `Session Band #1` 바깥으로 과하게 벌어졌는지 확인합니다.

핵심 해석은 아래처럼 보면 됩니다.

- `가격 > Session > 5-Day`이고 `20-Day VWAP`도 아래에서 받치면
  - 단기와 중기 평균 단가가 모두 아래에 있는 상태에 가깝습니다.
  - 눌림이 `Session VWAP` 근처에서 멈추는지 보는 흐름이 자연스럽습니다.

- `가격 < Session < 5-Day`이고 `20-Day VWAP`까지 위에 있으면
  - 짧은 기준부터 큰 기준까지 모두 위에서 눌리는 구조에 가깝습니다.
  - 반등이 `Session VWAP`를 회복하지 못하는지 보는 쪽이 더 자연스럽습니다.

- 가격이 `Session Band #1` 바깥으로 많이 벌어졌는데 `5-Day / 20-Day`와도 멀리 떨어져 있으면
  - 추세 지속보다 단기 과열/과매도 여부를 먼저 의심해볼 수 있습니다.
  - 바로 추격하기보다 `Session VWAP` 복귀 반응을 확인하는 편이 더 안전합니다.

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: `overlay=true`
- 지표명: `Advanced Multi Anchor VWAP`
- 기본 기준선: `Session`, `5-Day`, `20-Day`
- 기준 계산: `5-Day / 20-Day`는 `현재 일봉 포함 rolling daily window`
- 기본 선 색상: `Session=파랑`, `5-Day=주황`, `20-Day=보라`
- 기본 소스: `hlc3`
- 기본 밴드: `Session Band #1` 상/하단만 파란 점선으로 활성화, Fill은 기본 비활성
- 기본 밴드 계산: `Standard Deviation`

## 설정값을 어떻게 쓰면 좋은지

### 1. `Session / 5-Day / 20-Day`

- `Session`: 데이트레이딩에서 가장 빠르게 반응하는 기준입니다.
- `5-Day`: 이번 주 흐름에 가까운 단기 스윙 기준으로 보기 좋습니다.
- `20-Day`: 한 달 안팎의 더 큰 평균 단가 기준으로 보기 좋습니다.

핵심은 `오늘 평균 단가`와 `최근 며칠 평균 단가`가 같은 방향으로 정렬되는지 보는 것입니다.

- 단타면 `Session` 반응을 가장 먼저 보고
- 스윙 확인은 `5-Day`
- 큰 방향 확인은 `20-Day`

### 2. `Bands Calculation Mode`

- `Standard Deviation`: 실제 변동성에 따라 밴드 폭이 달라집니다.
- `Percentage`: 가격 대비 고정 비율로 밴드를 봅니다.

이 밴드는 `Session VWAP`에만 적용됩니다. 멀티 기준선 모두에 밴드를 붙이면 차트가 너무 복잡해져서, 가장 짧은 기준 하나에만 이격 해석을 집중하는 쪽이 실전에서는 더 낫습니다.

### 3. `Band #1 ~ #3`

- `1`: 가장 자주 닿는 기본 이격 구간
- `2`: 과열/과매도 해석이 강해지는 구간
- `3`: 추격보다 차익실현이나 역추세 경계가 필요한 구간

## 추천 사용 흐름

1. 가격이 `Session VWAP` 위에 있는지 아래에 있는지 먼저 봅니다.
2. `Session / 5-Day / 20-Day` 정렬이 같은 방향인지 확인합니다.
3. `Session VWAP` 밴드 바깥으로 과도하게 이격됐는지 봅니다.
4. 눌림이나 반등이 `Session VWAP` 또는 `Band #1` 안쪽으로 복귀하는지 확인한 뒤 진입을 판단합니다.

실전에서는 보통 아래처럼 해석합니다.

- `가격 > Session > 5-Day > 20-Day` 정렬이면 매수 우위를 우선 봅니다.
- `가격 < Session < 5-Day < 20-Day` 정렬이면 매도 우위를 우선 봅니다.
- `Session`만 강하고 `5-Day / 20-Day`가 못 따라오면 단기 반등 또는 단기 과열일 가능성을 열어둡니다.
- `Session VWAP`가 평평하면 추세보다 횡보 가능성을 우선 봅니다.
- `Session Band` 바깥 과도한 이격은 추격보다 되돌림 가능성을 먼저 봅니다.

## 세력 관점 해석

![멀티 앵커 VWAP 세력 관점 다이어그램](./smart-money-view.svg)

이 지표는 `오늘 평균 단가`, `최근 5일 평균 단가`, `최근 20일 평균 단가`가 어느 방향으로 정렬되는지로 시장 수용 방향을 읽는 데 가깝습니다.

### 1. `가격 > Session > 5-Day > 20-Day`

이 조합은 시장이 더 높은 가격을 `받아들이고 있다`는 뜻에 가깝습니다.

- 오늘 체결 평균이 단기와 중기 평균 위에서 유지되면, 당일 매수자가 손해 구간으로 바로 밀리지 않는 구조일 수 있습니다.
- `5-Day`와 `20-Day`가 아래에서 받치면, 단순 일중 반등보다 `며칠 단위 수용`으로 해석할 근거가 더 많아집니다.
- 이때 예상되는 움직임은 `Session 눌림 -> 재안착 -> 고점 재시험`입니다.

즉 세력 관점에서는 `짧은 평균 위에서 더 큰 평균까지 끌어올리는 단계`일 가능성을 먼저 봅니다.

### 2. `가격 < Session < 5-Day < 20-Day`

이 조합은 시장이 더 낮은 가격을 `정상 가격`으로 받아들이는 구간에 가깝습니다.

- Session 반등이 나와도 `5-Day` 아래에서 막히면, 위로 팔 물량이 계속 남아 있는 구조일 수 있습니다.
- `20-Day`까지 위에 있으면 단기 흔들림이 아니라 `더 큰 기준선도 모두 상단 공급`으로 해석할 수 있습니다.
- 이때 예상되는 움직임은 `짧은 반등 -> Session 재이탈 -> 저점 재시험`입니다.

즉 세력 관점에서는 `당일 평균마저 위에서 눌러가며 더 큰 평균 아래로 마크다운하는 단계`를 의심합니다.

### 3. `Session`만 먼저 꺾이거나 먼저 강해질 때

이 구간은 `주도권 전환 시도` 또는 `단기 함정` 후보입니다.

- Session만 먼저 `5-Day` 위로 올라가고 `20-Day`는 아직 아래로 못 넘으면, 짧은 숏 커버나 단기 반등일 수 있습니다.
- 반대로 Session만 먼저 약해지고 `5-Day / 20-Day`는 아직 위쪽 수용이면, 큰 추세 안의 일시적 눌림일 수 있습니다.
- 그래서 Session 신호 하나보다 `5-Day가 따라오는지`, `20-Day도 같은 방향인지`를 같이 봐야 합니다.

### 4. Rolling Daily Window의 의미

이 지표에서 `5-Day / 20-Day`는 고정 주봉/월봉이 아니라, `현재 일봉 포함 rolling daily window`입니다.

- 장중에도 현재 일봉 누적 체결이 계속 반영됩니다.
- 그래서 완전 고정된 상위 VWAP보다 `지금 며칠 평균이 어디로 움직이는지`를 더 빨리 볼 수 있습니다.
- 대신 장중에는 값이 조금씩 변하므로, 종가 기준 해석과 장중 해석을 구분해서 보는 편이 좋습니다.

### 5. 이 지표로 예측할 수 있는 움직임

- 정렬 유지: 눌림 뒤 추세 재개 가능성 증가
- Session 단독 이탈: 단기 흔들림 또는 주도권 전환 시도 가능성
- 5-Day 붕괴: 단기 스윙 수용 약화 가능성
- 20-Day 이탈: 더 큰 구조 전환 가능성 증가
- Session Band 과확장: 추세 지속보다 단기 평균 회귀 가능성 증가

## POC와 비교해서 보는 법

`VWAP`와 `POC`는 둘 다 공정가치를 보지만 역할이 다릅니다.

- `VWAP`는 해당 기간의 `평균 체결 단가`
- `POC`는 해당 프로파일 구간의 `가장 많이 거래된 단일 가격`

실전 해석은 이렇게 가져가면 단순합니다.

- `VWAP`는 방향과 평균 가격 기준
- `POC`는 거래가 가장 많이 모인 자리 기준
- `가격이 VWAP와 POC 위`면 롱 우위
- `가격이 VWAP와 POC 아래`면 숏 우위
- `가격이 VWAP와 POC 사이`면 전환 구간이라 애매한 경우가 많습니다.

이 스크립트는 `VWAP 기준선`을 보는 역할이므로, `POC`는 Volume Profile 계열 도구와 같이 보는 편이 좋습니다.

## 실전 해석 예시

### 롱 쪽으로 보기 좋은 상황

- `Session VWAP` 위에서 가격이 유지됩니다.
- `Session > 5-Day > 20-Day` 또는 최소한 `Session > 5-Day` 정렬이 나옵니다.
- `Session VWAP` 기울기가 완만하게라도 상승합니다.
- 눌림이 `Session VWAP` 또는 `Band #1` 부근에서 멈춥니다.
- 다음 봉에서 다시 `Session VWAP` 위로 종가가 회복됩니다.

이런 구조는 `평균 단가 위에서 눌림 매수` 시나리오로 보기 좋습니다.

### 숏 쪽으로 보기 좋은 상황

- `Session VWAP` 아래에서 가격이 유지됩니다.
- `Session < 5-Day < 20-Day` 또는 최소한 `Session < 5-Day` 정렬이 나옵니다.
- `Session VWAP` 기울기가 하락합니다.
- 반등이 `Session VWAP` 또는 `Band #1` 근처에서 막힙니다.
- 다음 봉에서 다시 `Session VWAP` 아래로 밀립니다.

이런 구조는 `평균 단가 아래에서 반등 매도` 시나리오로 보기 좋습니다.

### 평균 회귀로 보기 좋은 상황

- `Session VWAP`가 거의 평평합니다.
- 가격이 `Session Band #2` 또는 `Session Band #3`까지 과하게 벌어집니다.
- 직전 고점/저점 돌파가 실패하거나 꼬리가 길게 나옵니다.

이때는 추세 추종보다 `Session VWAP 복귀`를 목표로 보는 쪽이 더 자연스러울 수 있습니다.

## 실전 롱/숏 전략 예시

![멀티 앵커 VWAP 롱숏 전략 예시](./trade-strategy-example.svg)

### 롱 기준

- `롱 준비`: `가격 > Session > 5-Day > 20-Day` 정렬이 먼저 나옵니다.
- `롱 매수`: 첫 눌림이 `Session VWAP` 근처에서 멈추고 다시 Session 위로 안착할 때 진입을 준비합니다.
- `롱 매도`: 직전 고점 재시험 구간, `Session Band` 과확장 구간, 또는 둔화 신호가 겹치는 자리에서 분할 청산을 생각합니다.
- `무효화`: `Session` 아래 재이탈 뒤 `5-Day`까지 깨지면 시나리오를 약화시킵니다.

### 숏 기준

- `숏 준비`: `가격 < Session < 5-Day < 20-Day` 정렬이 먼저 나옵니다.
- `숏 매도`: 첫 반등이 `Session VWAP` 근처에서 막히고 다시 Session 아래로 밀릴 때 진입을 준비합니다.
- `숏 매수`: 직전 저점 재시험 구간, `Session Band` 과확장 구간, 또는 둔화 신호가 겹치는 자리에서 분할 청산을 생각합니다.
- `무효화`: `Session` 위 복귀 뒤 `5-Day`까지 회복하면 시나리오를 약화시킵니다.

### 이 지표를 쓸 때 중요한 점

- 이 지표는 `평균 단가 수용 구조`를 읽는 지표이지, 단독 진입 신호기는 아닙니다.
- 가장 좋은 자리는 `정렬 확인 후 첫 Session 반응`입니다.
- Session만 보지 말고 `5-Day / 20-Day`가 같은 방향으로 받치거나 눌리는지 꼭 같이 확인해야 합니다.

## 이 스크립트를 같이 보면 좋은 경우

- [`추세 추적`](../%EC%B6%94%EC%84%B8%20%EC%B6%94%EC%A0%81/README.md): 큰 방향을 먼저 정할 때
- [`거래량 압력 추적`](../%EA%B1%B0%EB%9E%98%EB%9F%89%20%EC%95%95%EB%A0%A5%20%EC%B6%94%EC%A0%81/README.md): 눌림/반등 자리에 실제 체결 압력이 붙는지 확인할 때
- [`MACD 다이버전스 추적`](../MACD/README.md): 평균 단가 기준 위아래에서 모멘텀 재가속을 볼 때

즉:

- `추세 추적`은 방향
- `Advanced Multi Anchor VWAP`는 세션/5일/20일 평균 단가 기준
- `거래량`은 실제 힘
- `MACD`는 힘 재개

## 주의사항

- `VWAP`는 거래량 데이터가 필요하므로, 거래량이 없는 종목에서는 동작하지 않습니다.
- `Session / 5-Day / 20-Day`를 모두 켜면 차트가 복잡해질 수 있으니, 본인 매매 호흡에 맞는 선만 남겨두는 편이 좋습니다.
- `5-Day / 20-Day`는 `현재 진행 중인 일봉` 누적치까지 포함해서 계산됩니다.
- 세션이 바뀌는 지점에서는 `Session VWAP`와 밴드만 끊어지고, `5-Day / 20-Day`는 rolling 방식으로 이어집니다.
- `Hide VWAP on 1D or Above`를 켜면 일봉 이상에서는 보이지 않습니다.
- 이 지표 하나만으로 진입을 확정하기보다, 추세 구조와 거래량 반응을 같이 보는 편이 더 안전합니다.
