# 이평선 리본

트레이딩뷰용 Pine Script 오버레이 지표 설명서입니다.

대상 스크립트:
- [`moving-average-ribbon.pine`](./moving-average-ribbon.pine)

이 지표는 `20, 25, 30, 35, 40, 45, 50` 이동평균을 리본처럼 겹쳐 보여 주고, `SMA / EMA / WMA / VWMA` 중 선택한 방식으로 전체 리본을 계산합니다. 리본 구조 변화에 맞춰 아래 4가지 라벨을 캔들 상단에 표시합니다.

## 라벨 정의

| 라벨 | 현재 판정 방식 |
| --- | --- |
| `리본 상승 전환` | 짧은 이평선부터 긴 이평선까지 순차적으로 위에 정렬되고, 옵션에 따라 전부 우상향 기울기일 때 |
| `리본 하락 전환` | 짧은 이평선부터 긴 이평선까지 순차적으로 아래에 정렬되고, 옵션에 따라 전부 우하향 기울기일 때 |
| `리본 지지` | 상승 정렬 상태에서 가격이 리본 상단을 눌렀다가 다시 리본 위로 마감할 때 |
| `리본 저항` | 하락 정렬 상태에서 가격이 리본 하단을 찔렀다가 다시 리본 아래로 마감할 때 |

## 설정

- `이동평균 종류`: `SMA`, `EMA`, `WMA`, `VWMA` 중 하나를 선택합니다.
- `MA 100`, `MA 200`: 장기 기준선과 두 선 사이 배경 채우기 표시 여부를 정합니다.

## 해석 팁

- `리본 상승 전환`, `리본 하락 전환`은 방향 전환 확인용입니다.
- `리본 지지`, `리본 저항`은 이미 정렬된 리본 위에서 나오는 재테스트 확인용입니다.
- 기본값은 `봉마감 후 라벨 확정`이 켜져 있어서, 진행 중인 봉에서 라벨이 잠깐 나타났다가 사라지는 현상을 줄입니다.
- `지지/저항 리본 관통 허용 비율`을 낮추면 더 얕은 눌림만 잡고, 높이면 리본 안쪽 깊은 테스트도 허용합니다.
- 모든 라벨은 캔들 상단에 표시되므로, 실제 매수/매도 방향보다는 `구조 이름을 읽는 용도`에 가깝습니다.
