# 이평선 리본

트레이딩뷰용 Pine Script 오버레이 지표 설명서입니다.

대상 스크립트:
- [`moving-average-ribbon.pine`](./moving-average-ribbon.pine)

이 지표는 `20, 25, 30, 35, 40, 45, 50` 이동평균을 리본처럼 겹쳐 보여 주고(`SMA / EMA / WMA / VWMA` 중 선택), `MA 100 / 200` 장기선을 함께 그립니다. 표시는 `횡보 압축`(보라색 원) 하나만 두며, 이치모쿠 메인 신호의 **신뢰도 필터(휩쏘 경고)** 용도로 씁니다.

## 표시 정의

| 표시 | 현재 판정 방식 | 모양 |
| --- | --- | --- |
| `횡보 압축` | 리본 폭(`MA20~50`의 최댓값−최솟값)이 `ATR × 압축 배수`보다 좁아지는 **압축 구간으로 처음 진입한 봉** | 진한 보라색 원 (캔들 아래) |

- 압축에 진입한 **첫 봉**에만 보라색 원이 1회 찍힙니다(압축이 이어지는 동안 중복 표시되지 않음).

## 이치모쿠 신뢰 활용

이치모쿠는 **추세추종**이라 횡보장에서 휩쏘에 약합니다. 리본이 좁게 압축됐다는 것은 이동평균들이 한데 뭉쳐 **방향이 사라진 횡보 구간**이라는 뜻이고, 이 자리에서 나오는 이치모쿠 매수/매도는 휩쏘일 가능성이 큽니다.

| 상황 | 해석 |
| --- | --- |
| 이치모쿠 신호 + `횡보 압축`(보라색 원) 이후 압축이 이어지는 구간 | **휩쏘 의심 → 관망**, 또는 다른 보조지표 정렬을 더 까다롭게 확인 |
| 이치모쿠 신호 + 리본이 압축이 아닌(벌어진) 구간 | 추세가 살아 있을 확률이 커 **신호 신뢰에 무리 없음** |

- 리본은 후행성이 있으므로 **진입 트리거가 아니라 신뢰도 필터**입니다. 이치모쿠가 신호를 만들고, 리본은 "지금이 휩쏘 구간인지"만 걸러 줍니다.

## 설정

- `이동평균 종류`: `SMA`, `EMA`, `WMA`, `VWMA` 중 하나를 선택합니다.
- `MA 100`, `MA 200`: 장기 기준선과 두 선 사이 배경 채우기 표시 여부를 정합니다.
- `횡보 압축 경고 라벨`: 라벨 표시를 켜고 끕니다.
- `봉마감 후 라벨 확정`: 켜면 봉이 마감된 뒤에만 라벨을 확정합니다.
- `리본폭 ATR 기준`: 압축 판정에 쓰는 ATR 길이입니다.
- `압축(휩쏘) 판정 ATR 배수`: 낮출수록 더 좁을 때만(엄격하게) 압축으로 보고, 높일수록 더 자주 압축으로 봅니다.

## 해석 팁

- `횡보 압축`은 방향을 만드는 신호가 아니라 `이치모쿠 신호를 믿지 말아야 할 구간`을 알려 주는 용도입니다.
- 보라색 원이 찍힌 뒤 리본이 다시 벌어지기 시작하면, 그 방향으로 추세가 살아날 준비 구간으로 봅니다.
- 기본값은 `봉마감 후 라벨 확정`이 켜져 있어서, 진행 중인 봉에서 라벨이 잠깐 나타났다가 사라지는 현상을 줄입니다.
