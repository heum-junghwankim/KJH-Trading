# 이평선 리본

트레이딩뷰용 Pine Script 오버레이 지표 설명서입니다.

대상 스크립트:
- [`moving-average-ribbon.pine`](./moving-average-ribbon.pine)

이 지표는 `20, 25, 30, 35, 40, 45, 50` 이동평균을 리본처럼 겹쳐 보여 주고(`SMA / EMA / WMA / VWMA` 중 선택), `MA 100 / 200` 장기선을 함께 그립니다. 리본의 **압축 → 분출** 흐름을 캔들 아래 다이아몬드로 표시하며, 이치모쿠 메인 신호의 **신뢰도 필터(휩쏘 경고 + 분출 확인)** 용도로 씁니다.

## 표시 정의

| 표시 | 판정 방식 | 모양 |
| --- | --- | --- |
| `횡보 압축` | 리본 폭(`MA20~50`의 최댓값−최솟값)이 `ATR × 압축 배수`보다 좁아지는 **압축 구간으로 처음 진입한 봉** | 회색 다이아몬드 (캔들 아래) |
| `상승 분출` | 압축이 풀리는(폭이 다시 벌어지는) 첫 봉에 `MA20 ≥ MA50`(상승 방향) | 빨간 다이아몬드 (캔들 아래) |
| `하락 분출` | 압축이 풀리는 첫 봉에 `MA20 < MA50`(하락 방향) | 파란 다이아몬드 (캔들 아래) |

- 모든 표시는 상태가 새로 바뀐 **첫 봉**에만 1회 찍힙니다.
- `상승 분출 / 하락 분출`은 **반드시 `횡보 압축` 뒤에만** 나옵니다.

## 상태 흐름 (압축 → 분출)

`압축 → 분출 → 압축 → 분출 …` 순서로만 진행하는 상태머신입니다.

1. 리본 폭이 `ATR × 압축 배수`보다 좁아지면 `횡보 압축`(회색 다이아몬드) — 동시에 다음 분출을 **무장(armed)** 시킵니다.
2. 압축이 풀려 폭이 다시 벌어지는 **첫 봉**에, 그 순간 `MA20`과 `MA50`의 위치로 방향을 정해 분출 1회:
   - `MA20 ≥ MA50` → `상승 분출`(빨간 다이아몬드)
   - `MA20 < MA50` → `하락 분출`(파란 다이아몬드)
   - 분출이 나오면 **무장 해제**됩니다.
3. 한 번 분출이 나오면, **다시 `횡보 압축`이 와야** 다음 분출(상승이든 하락이든)이 허용됩니다. 즉 분출이 연속으로 나오지 않습니다.

> `분출`은 "압축으로 응축된 에너지가 어느 방향으로 터졌는지"를 한 번 찍어 주는 표시입니다. 방향 판정은 압축 해소 시점의 **빠른선(MA20) vs 느린선(MA50)** 으로 단순화했습니다.

## 이치모쿠 신뢰 활용

이치모쿠는 **추세추종**이라 횡보장에서 휩쏘에 약합니다. 리본 압축은 이동평균들이 한데 뭉쳐 **방향이 사라진 횡보 구간**을, 분출은 그 횡보가 **어느 방향으로 풀렸는지**를 알려 줍니다.

| 상황 | 해석 |
| --- | --- |
| 이치모쿠 신호 + `횡보 압축` 이후 압축이 이어지는 구간 | **휩쏘 의심 → 관망**, 또는 다른 보조지표 정렬을 더 까다롭게 확인 |
| 이치모쿠 매수 + `상승 분출` | 횡보가 위로 풀림 → **롱 신뢰↑** (압축에서 갓 풀린 추세 초입) |
| 이치모쿠 매도 + `하락 분출` | 횡보가 아래로 풀림 → **숏 신뢰↑** |
| 이치모쿠 신호 방향과 분출 방향이 반대 | 구조와 신호 충돌 → **신뢰 낮음** |

- 리본은 후행성이 있으므로 **진입 트리거가 아니라 신뢰도 필터**입니다. 이치모쿠가 신호를 만들고, 리본은 "지금이 휩쏘 구간인지 / 어느 방향으로 분출했는지"를 걸러 줍니다.
- `횡보 압축`은 이치모쿠 `기준선(22)`이 평평해지는 구간과 대체로 겹칩니다. 이때 전환선·기준선 크로스가 휩쏘일 확률이 커, 압축 구간 신호는 관망이 안전합니다.

## 설정

- `이동평균 종류`: `SMA`, `EMA`, `WMA`, `VWMA` 중 하나를 선택합니다.
- `MA 100`, `MA 200`: 장기 기준선과 두 선 사이 배경 채우기 표시 여부를 정합니다.
- `횡보 압축 라벨`: 횡보 압축(회색 다이아몬드) 표시를 켜고 끕니다.
- `분출(상승/하락) 라벨`: 상승/하락 분출 다이아몬드 표시를 켜고 끕니다.
- `봉마감 후 라벨 확정`: 켜면 봉이 마감된 뒤에만 표시를 확정합니다.
- `리본폭 ATR 기준`: 압축 판정에 쓰는 ATR 길이입니다.
- `압축(휩쏘) 판정 ATR 배수`: 낮출수록 더 좁을 때만(엄격하게) 압축으로 보고, 높일수록 더 자주 압축으로 봅니다. (분출 시점도 이 기준을 다시 넘는 순간으로 정해집니다.)

## 해석 팁

- `횡보 압축`(회색)은 방향을 만드는 신호가 아니라 `이치모쿠 신호를 믿지 말아야 할 구간`을 알려 줍니다.
- `상승 분출`(빨강) / `하락 분출`(파랑)은 그 압축이 **어느 쪽으로 풀렸는지**를 알려 주는 추세 출발 후보입니다. 같은 방향 이치모쿠 신호와 겹치면 신뢰가 큽니다.
- 분출은 압축당 1회만 나오므로, 분출 뒤 추세가 이어지는지는 가격·거래량으로 별도 확인합니다.
- 기본값은 `봉마감 후 라벨 확정`이 켜져 있어서, 진행 중인 봉에서 표시가 잠깐 나타났다가 사라지는 현상을 줄입니다.
