# PineScript 트레이딩 가이드

이 폴더는 현재 저장소에 있는 Pine Script 지표를 `같은 해석 흐름`으로 읽기 위한 상위 안내서입니다.

지금 기준 메인 축은 [`비정상 가격 추적 (캔들)`](./비정상%20가격%20추적%20(캔들)/README.md)이고, 나머지 지표는 그 판단을 보완하는 역할입니다.

## 한눈에 보기

| 지표 | 한 줄 역할 |
| --- | --- |
| [`비정상 가격 추적 (캔들)`](./비정상%20가격%20추적%20(캔들)/README.md) | 자리, 감시, 기준 봉/후보, 스윕, 리포트를 봅니다. |
| [`Auto VWAP`](./VWAP/README.md) | 평균 체결 단가와 확장 구간을 봅니다. |
| [`거래량 압력 추적`](./거래량%20압력%20추적/README.md) | 현재 봉에 실제 압력이 붙는지 봅니다. |
| [`MACD 다이버전스 추적`](./MACD/README.md) | 내부 모멘텀이 다시 붙는지 봅니다. |
| [`CCI 익스트림 크로스 시그널`](./CCI/README.md) | 극단값 이후 재진입 크로스를 봅니다. |
| [`비정상 가격 추적 (보조)`](./비정상%20가격%20추적%20(보조)/README.md) | 추격 위험과 반대 방향 반전 위험을 거릅니다. |

짧게 줄이면:

- `캔들`은 자리
- `VWAP`은 기준 단가
- `거래량`은 압력
- `MACD`는 힘 재개
- `CCI`는 극단값 뒤 재진입
- `보조`는 위험 필터

## 추천 읽는 순서

1. [`비정상 가격 추적 (캔들)`](./비정상%20가격%20추적%20(캔들)/README.md)로 현재 자리와 리포트 `포지션`을 먼저 봅니다.
2. [`Auto VWAP`](./VWAP/README.md)으로 가격이 기준 단가 위인지 아래인지 확인합니다.
3. [`거래량 압력 추적`](./거래량%20압력%20추적/README.md)으로 실제 압력이 같은 방향인지 확인합니다.
4. [`CCI 익스트림 크로스 시그널`](./CCI/README.md)로 과열/과매도 뒤 재진입이 나오는지 봅니다.
5. [`MACD 다이버전스 추적`](./MACD/README.md)으로 모멘텀 재개 여부를 봅니다.
6. [`비정상 가격 추적 (보조)`](./비정상%20가격%20추적%20(보조)/README.md)로 반대 방향 위험이 없는지 마지막으로 체크합니다.

## 가장 단순한 해석

| 상황 | 보통 이렇게 읽습니다 |
| --- | --- |
| 롱 쪽 | 캔들 자리 형성 + VWAP 회복 + 매수 압력 + MACD 재가속 + 반대 위험 약함 |
| 숏 쪽 | 캔들 자리 형성 + VWAP 재이탈 + 매도 압력 + MACD 재하락 + 반대 위험 약함 |
| 관망 | 자리, 압력, 모멘텀 중 하나라도 비어 있거나 서로 충돌 |

## 관망이 더 나은 경우

- 메인 캔들 신호는 좋은데 거래량이 약할 때
- VWAP을 중심으로 계속 왕복할 때
- MACD가 아직 애매한데 먼저 진입하고 싶어질 때
- 보조 지표에서 반대 방향 `H`가 뜰 때
- 한두 지표만 좋고 나머지가 비어 있을 때

핵심은 `좋은 진입을 더 찾는 것`보다 `나쁜 진입을 피하는 것`입니다.

## 마지막 원칙

- 메인 지표보다 먼저 보조 지표로 진입하지 않기
- 거래량 없이 모양만 좋은 신호는 한 번 더 의심하기
- 기준 단가와 반대 방향으로 무리하게 추격하지 않기
- 확정봉 기준 신호라는 점을 항상 같이 보기
